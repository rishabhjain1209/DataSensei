Python Try Except Blocks

1. Introduction

Exception handling is a crucial aspect of programming, allowing you to gracefully handle errors that may occur during the execution of your code. Python provides the try, except, else, and finally blocks for effective exception handling.

2. Try Block

The try block is used to enclose a block of code where you expect an exception might occur. It allows you to test a piece of code for errors.


try:
    # Code that may raise an exception
except:
    # Code to handle the exception

Example:


try:
    x = 10 / 0  # This will raise a ZeroDivisionError
except ZeroDivisionError:
    print("Division by zero error")

3. Multiple Except Blocks

You can have multiple except blocks to handle different types of exceptions. Python will execute the appropriate block based on the type of exception that occurs.

Example:


try:
    x = int("abc")  # This will raise a ValueError
except ValueError:
    print("Invalid conversion to integer")
except ZeroDivisionError:
    print("Division by zero error")

4. Else Block

The else block is executed if no exceptions are raised in the try block. It is often used for code that should run only when there are no exceptions.

Example:


try:
    x = 10 / 2
except ZeroDivisionError:
    print("Division by zero error")
else:
    print("No exceptions occurred")

5. Finally Block

The finally block is always executed, regardless of whether an exception was raised or not. It is useful for cleanup operations such as closing files or releasing resources.

Example:


try:
    x = 10 / 2
except ZeroDivisionError:
    print("Division by zero error")
finally:
    print("This will always execute")

6. Raising Exceptions

You can explicitly raise an exception using the raise keyword. This is useful when you want to trigger an exception based on certain conditions.

Example:


x = -1
if x < 0:
    raise Exception("Sorry, no numbers below zero")


x = "hello"
if not isinstance(x, int):
    raise TypeError("Only integers are allowed")

Exception handling is a fundamental part of writing robust and reliable Python programs. By using try, except, else, and finally blocks, you can gracefully handle errors and ensure that your code behaves as expected even when exceptions occur.



User Input in Python:

Python allows you to take user input using the input() function in Python 3.6 and later versions. In Python 2.7, you can use raw_input() for the same purpose. Here's an example of how to use input() in Python 3.6:

# Python 3.6
username = input("Enter username:")
print("Username is: " + username)

In this code, the input() function is used to prompt the user to enter their username. The input provided by the user is stored in the username variable, and it's then printed to the screen.

In Python 2.7, you would use raw_input() instead of input().

String Formatting in Python:

Basic String Formatting:

String formatting allows you to create dynamic strings by replacing placeholders with actual values. Python provides the format() method for this purpose. You can use curly braces {} as placeholders in a string and then use the format() method to replace those placeholders with values. Here's a basic example:

price = 49
txt = "The price is {} dollars"
formatted_text = txt.format(price)
print(formatted_text)

Output:

The price is 49 dollars

Formatting with Precision:

You can also format values with precision using :.2f to display numbers with two decimal places:

txt = "The price is {:.2f} dollars"
formatted_text = txt.format(49)
print(formatted_text)

Output:

The price is 49.00 dollars

Multiple Values and Placeholders:

You can use multiple placeholders and provide corresponding values in the format() method:

quantity = 3
itemno = 567
price = 49
myorder = "I want {} pieces of item number {} for {:.2f} dollars."
formatted_order = myorder.format(quantity, itemno, price)
print(formatted_order)

Output:

I want 3 pieces of item number 567 for 49.00 dollars.

Index Numbers and Reusing Values:

You can use index numbers inside curly braces to specify the order of values to be replaced:

age = 36
name = "John"
txt = "His name is {1}. {1} is {0} years old."
formatted_text = txt.format(age, name)
print(formatted_text)

Output:

His name is John. John is 36 years old.

Named Indexes:

You can use named indexes by providing names inside curly braces and then passing named arguments to the format() method:

myorder = "I have a {carname}, it is a {model}."
formatted_order = myorder.format(carname="Ford", model="Mustang")
print(formatted_order)

Output:

I have a Ford, it is a Mustang.

These are some of the fundamental concepts of user input and string formatting in Python. They allow you to interact with users and create formatted, dynamic strings in your Python programs.

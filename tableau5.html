Lesson 9: Calculations and Expressions

  1. Introduction to Calculated Fields

    Calculated fields in Tableau allow you to create new fields by performing calculations on existing data. They are powerful for creating custom metrics and transforming data.

    To create a calculated field:

    • Right-click in the "Data" pane and select "Create Calculated Field."
    • Enter a formula or expression.

    Example Code for Creating a Calculated Field:

                1. Right-click in the "Data" pane and select "Create Calculated Field."
            2. Name your calculated field (e.g., "Profit Margin").
            3. Enter the formula: (SUM([Profit]) / SUM([Sales])) * 100
            4. Click OK to create the calculated field.

  2. Using Functions and Formulas

    Tableau provides a wide range of functions and formulas for calculated fields. These include mathematical, string, date, logical, and table calculations.

    Example Code for Using a Function in a Calculated Field:

                1. Create a calculated field.
            2. Name it (e.g., "Year of Order Date").
            3. Use the DATEPART() function: DATEPART('year', [Order Date])

  3. Creating Custom Calculations

    Calculated fields can be customized to meet specific analytical needs. You can use IF statements, logical operators, and nesting to create complex calculations.

    Example Code for Creating a Custom Calculation:

                1. Create a calculated field.
            2. Name it (e.g., "Customer Segment Classification").
            3. Use an IF statement: IF [Profit] > 1000 THEN 'High Profit'
                                   ELSEIF [Profit] > 0 THEN 'Profit'
                                   ELSE 'Loss'
                                   END

Lesson 10: Parameters and Actions

  1. Understanding Parameters

    Parameters in Tableau allow users to input a value that can be used in calculations, filters, and dynamic actions.

    To create a parameter:

    • Right-click in the "Data" pane and select "Create Parameter."
    • Define the parameter properties, including data type, allowable values, and current value.

    Example Code for Creating a Parameter:

                1. Right-click in the "Data" pane and select "Create Parameter."
            2. Name your parameter (e.g., "Discount Percentage").
            3. Choose the data type (e.g., float).
            4. Define allowable values (e.g., between 0 and 100).

  2. Implementing Actions for User Interactivity

    Actions in Tableau enable interactive features by allowing users to interact with one visualization to affect another.

    To create an action:

    • Go to the "Dashboard" tab and select "Actions."
    • Choose the type of action (e.g., filter, highlight, or URL).
    • Define the source sheet and target sheet.

    Example Code for Implementing a Filter Action:

                1. Create a dashboard.
            2. Go to the "Dashboard" tab and select "Actions."
            3. Click "Add Action" and choose "Filter."
            4. Define the source sheet (e.g., a bar chart) and the target sheet (e.g., a map).
            5. Set the field to filter and configure the action as needed.

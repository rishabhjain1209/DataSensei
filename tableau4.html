Lesson 7: Maps and Geographic Visualizations

  1. Creating Maps in Tableau

    Tableau is a powerful tool for visualizing geographic data. To create maps:

    • Drag a geographic dimension (e.g., Country) to the "Rows" or "Columns" shelf.
    • Tableau will automatically recognize the geographic data and convert it into a map.

    Example Code for Creating a Map:

                1. Connect to your data source containing geographic information (e.g., latitude and longitude).
            2. Drag "City" to the "Rows" shelf and "Sales" to the "Color" shelf.
            3. Tableau will generate a map showing sales by city.

  2. Working with Geographic Data

    Tableau can work with various types of geographic data, including latitude and longitude coordinates, country names, and postal codes. You can also customize geographic roles, such as setting a field as a geographic dimension or attribute.

  3. Adding Layers and Customizing Maps

    To enhance your geographic visualizations:

    • You can add additional layers to your maps, such as background images or reference lines.
    • Customize map elements, including colors, shapes, and tooltips.

Lesson 8: Building Dashboards

  1. Design Principles for Dashboards

    Effective dashboard design follows key principles:

    • Clarity: Ensure that information is presented clearly and concisely.
    • Consistency: Maintain a consistent layout and color scheme.
    • Prioritization: Highlight the most important insights.
    • Interactivity: Make dashboards interactive for user engagement.

  2. Combining Multiple Visualizations into a Dashboard

    To create a dashboard in Tableau:

    • Select the worksheets you want to include.
    • Go to the "Dashboard" tab and click "New Dashboard."
    • Drag and drop worksheets onto the dashboard canvas.
    • Arrange and resize them as needed.

    Example Code for Creating a Dashboard:

                1. Create multiple visualizations (e.g., bar chart, line chart, and map).
            2. Go to the "Dashboard" tab.
            3. Click "New Dashboard."
            4. Drag and drop your visualizations onto the dashboard canvas.
            5. Arrange and customize them to create an interactive dashboard.

  3. Interactivity and Filtering in Dashboards

    You can add interactivity to dashboards using filters, actions, and parameters:

    • Filters: Allow users to filter data by selecting specific criteria.
    • Actions: Create dynamic interactions between different visualizations within the dashboard.
    • Parameters: Enable users to change parameters to adjust calculations or control visual elements.

    Example Code for Adding a Filter to a Dashboard:

                1. Create a dashboard.
            2. Drag a field (e.g., Region) to the "Filters" shelf.
            3. Users can now select regions to filter the entire dashboard.

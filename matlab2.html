Module 2: MATLAB Programming Fundamentals

Writing and Executing MATLAB Scripts:

In MATLAB, scripts are files that contain a sequence of MATLAB commands. Here's how to create and execute a MATLAB script:

  1. Create a Script:

    Open the MATLAB Editor or Script Window, then type your MATLAB commands. Save the file with a .m extension, e.g., myscript.m.

  2. Run a Script:

    To run the script, simply type its name (without the file extension) in the Command Window and press Enter:

    myscript

    The MATLAB interpreter will execute the commands in the script sequentially.

In MATLAB, you can create variables to store data. Variable names are case-sensitive and must begin with a letter, followed by letters, numbers, or underscores. Here's how to define variables:


x = 5; % Assigns the value 5 to the variable x
name = 'John'; % Assigns the string 'John' to the variable name

You can use the whos command to display information about variables in the workspace:

whos

Conditional Statements (if-else):

MATLAB supports conditional statements for making decisions in your code. The basic syntax for an if-else statement is:


if condition
    % Code to execute if the condition is true
else
    % Code to execute if the condition is false
end

For example:


grade = 75;
if grade >= 60
    disp('Pass');
else
    disp('Fail');
end

Loops (for and while):

Loops in MATLAB allow you to repeat a block of code multiple times. MATLAB supports both for and while loops.

For Loops:

A for loop is used when you know the number of iterations in advance. The basic syntax is:


for variable = start:step:end
    % Code to execute in each iteration
end

For example, a loop to print the numbers from 1 to 5:


for i = 1:5
    disp(i);
end

While Loops:

A while loop is used when you want to repeat a block of code until a condition becomes false. The basic syntax is:


while condition
    % Code to execute as long as the condition is true
end

For example, a while loop to count down from 5 to 1:


count = 5;
while count > 0
    disp(count);
    count = count - 1;
end

User-Defined Functions:

You can create your own custom functions in MATLAB using the function keyword. Here's the basic structure of a user-defined function:


function output = functionName(input1, input2)
    % Function code here
    output = result; % Optional: specify the function's output
end

For example, a simple function to calculate the square of a number:


function result = square(x)
    result = x^2;
end

You can then call this function from your script or the Command Window:


result = square(5); % Calls the square function and assigns the result to the variable "result"

These fundamental concepts in MATLAB programming—scripts, variables, conditional statements, loops, and user-defined functions—form the building blocks for creating more complex programs and solving a wide range of numerical and scientific computing problems.

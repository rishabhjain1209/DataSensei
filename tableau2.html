Lesson 3: Data Import and Data Types

  1. Importing Data into Tableau

    Tableau offers several methods for importing data:

    • Connect to Data Source: As discussed in the previous lesson, you can connect to various data sources, including files, databases, and online sources.
    • Data Extracts: Tableau can create data extracts, which are optimized subsets of data for faster performance and offline use.
    • Copy and Paste: You can also copy data from spreadsheets and paste it directly into Tableau.

  2. Understanding Data Types

    Data in Tableau is classified into various data types, including:

    • Dimension: Categorical data, such as names, categories, or dates.
    • Measure: Numeric data that can be aggregated, like sales, profits, or quantities.
    • Continuous: Numeric data that can take any value within a range (e.g., temperature).
    • Discrete: Numeric data that consists of distinct, separate values (e.g., number of customers).

  3. Cleaning and Formatting Data

    Data often requires cleaning and formatting to be useful. Tableau provides tools to:

    • Remove Null Values: Eliminate rows with missing data.
    • Rename Fields: Make field names more descriptive.
    • Create Calculated Fields: Generate new fields based on existing data.
    • Split and Merge Fields: Divide or combine data within fields.

Lesson 4: Data Joins and Blending

  1. Joining Data from Multiple Sources

    Tableau allows you to combine data from different sources using joins. Common types of joins include:

    • Inner Join: Returns only the matching rows from both tables.
    • Left Join (Outer Join): Returns all rows from the left table and matching rows from the right table.
    • Right Join (Outer Join): Returns all rows from the right table and matching rows from the left table.
    • Full Outer Join: Returns all rows from both tables, including non-matching rows.

    Example Code for Joining Two Excel Sheets:

                1. Connect to your first Excel data source.
            2. Drag a second Excel data source onto the canvas.
            3. Define the join conditions by dragging fields from one source to another (e.g., linking a common customer ID).
            4. Choose the join type (inner, left, right, or full outer).
            5. Click "Add" to add the join clause.

  2. Data Blending for Combining Data

    Data blending is used when you have data from different data sources (e.g., Excel and a database) and you want to combine them without a traditional database join. It's often used when joins are not feasible due to data source limitations.

    Example Code for Data Blending:

                1. Connect to your primary data source (e.g., Excel).
            2. Drag a secondary data source (e.g., a database) onto the canvas.
            3. Define relationships by matching fields from both sources (e.g., customer ID).
            4. Tableau will automatically create a data blend relationship.
            5. Use fields from both sources in your visualization.

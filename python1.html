Python Overview

Python is a widely-used, versatile, and high-level programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991. Python is highly popular and finds application in various domains, including web development (server-side scripting), software development, mathematics, and system scripting. Let's dive into more details and provide a code example to illustrate Python's capabilities.

What Python Can Do?

  1. Web Development (Server-Side): Python can be used to create web applications. It offers various web frameworks such as Django and Flask that simplify web development tasks.
  2. Software Development: Python is utilized in software development for creating desktop applications, command-line tools, and more. Its extensive standard library provides modules for various tasks, reducing the need for reinventing the wheel.
  3. Mathematics: Python is a popular choice for mathematical and scientific computing. Libraries like NumPy and SciPy make complex mathematical operations and data manipulation efficient.
  4. System Scripting: Python serves as a powerful scripting language for automating system administration tasks, file manipulation, and interacting with the operating system.
  5. Database Interaction: Python can connect to various database systems using libraries like SQLAlchemy or interact with databases directly through modules like SQLite.
  6. Big Data and Complex Mathematics: Python's data analysis and visualization libraries (e.g., Pandas, Matplotlib) make it suitable for handling big data and performing data analysis tasks.
  7. Rapid Prototyping and Production-Ready Software: Python is well-suited for rapid prototyping due to its clean syntax and extensive library support. Projects developed in Python can easily transition to production-ready software.

Why Choose Python?

  1. Cross-Platform Compatibility: Python runs on various platforms, including Windows, Mac, Linux, and even on small devices like Raspberry Pi, ensuring code portability.
  2. Simple and Readable Syntax: Python's syntax resembles the English language, making it easy for developers to read and write code. This simplicity reduces the learning curve for beginners.
  3. Concise Code: Python emphasizes code readability and allows developers to achieve more with fewer lines of code compared to some other programming languages.
  4. Interpreted Language: Python is an interpreted language, which means code can be executed immediately after writing, facilitating rapid prototyping and development.
  5. Versatile Paradigms: Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming, giving developers flexibility in coding styles.

Python Syntax vs. Other Languages

  • Python uses new lines to complete a command instead of semicolons or parentheses commonly used in other languages.
  • Python relies on indentation (whitespace) to define scope, such as loops, functions, and classes, while other languages use curly braces for this purpose.

Code Example:

Here's a simple Python code example that prints "Hello, World!" to the console, illustrating Python's readability and simplicity:

# This is a Python comment
        print("Hello, World!")

To execute this code, you can save it to a .py file (e.g., hello.py) and run it using the Python interpreter:

python hello.py

This will display "Hello, World!" on your console, showcasing how Python's straightforward syntax allows you to achieve tasks with minimal code.



Python Installation and Quickstart

Python Installation

Python is a versatile programming language that may already be installed on your computer. To check if you have Python installed, follow these steps:

On Windows:

  1. Open the Command Line (cmd.exe).
  2. Run the following command:
C:\Users\Your Name>python --version

On Linux or Mac:

  1. Open the Terminal.
  2. Run the following command:
python --version

If Python is installed, you will see its version number displayed. If it's not installed, you can download and install Python for free from the official website: https://www.python.org/.

Python Quickstart

Python is an interpreted programming language, which means you write Python (.py) files in a text editor and then execute them using the Python interpreter. Here's how to run a Python file from the command line:

C:\Users\Your Name>python helloworld.py

In the above command, replace "helloworld.py" with the name of your Python file. Let's create a simple Python file called helloworld.py in any text editor:

helloworld.py

```python
        print("Hello, World!")
        ```

Save your file. Open the command line and navigate to the directory where you saved your file. Then, run the following command:

C:\Users\Your Name>python helloworld.py

The output should be:

Hello, World!

Congratulations, you have successfully written and executed your first Python program.

Python Command Line

You can test short snippets of Python code directly in the Python command line interface without writing code in a file. Here's how to access the Python command line on Windows, Mac, or Linux:

On Windows:

Open the Command Line (cmd.exe) and run:

C:\Users\Your Name>python

Or, if "python" doesn't work, you can try "py":

C:\Users\Your Name>py

From there, you can write and execute Python code, including the "Hello, World!" example from earlier in the tutorial:

>>> print("Hello, World!")

This will display "Hello, World!" in the command line.

To exit the Python command line interface, simply type the following:

>>> exit()

This will return you to the regular command prompt. The Python command line is useful for testing code snippets and exploring Python interactively.



Python Syntax, Indentation, and Variables

Execute Python Syntax

As we learned in the previous page, Python syntax can be executed in two primary ways:

  1. Interactive Mode (Command Line):
    • You can write Python code directly in the Command Line or Python interpreter prompt (>>>) and execute it immediately.
    >>> print("Hello, World!")

    This will output: Hello, World!

  2. Script Mode (Python Files):
    • Create a Python script file with the .py file extension (e.g., myfile.py).
    • Run the script in the Command Line using the python command:
    C:\Users\Your Name>python myfile.py

    This will execute the Python code saved in myfile.py.

Python Indentation

Indentation in Python refers to the spaces or tabs at the beginning of a code line. Unlike many other programming languages where indentation is for readability only, in Python, it is crucial for defining the structure of your code.

Python uses indentation to indicate a block of code. For example:

if 5 > 2:
            print("Five is greater than two!")

In the above code, the indentation with four spaces before print signifies that it is part of the if block. If you skip the indentation, Python will raise an error:

# Syntax Error
        if 5 > 2:
        print("Five is greater than two!")

The number of spaces for indentation is flexible, but it must be consistent within the same block of code. Common practice is to use four spaces for each level of indentation:

if 5 > 2:
            print("Five is greater than two!")
            if 4 < 3:
                print("This is another block of code.")

However, you can use tabs or a different number of spaces as long as you are consistent:

if 5 > 2:
          print("Five is greater than two!")  # Using two spaces for indentation
        if 5 > 2:
                print("Five is greater than two!")  # Using eight spaces for indentation

But mixing indentation styles within the same block of code will result in an error:

# Syntax Error
        if 5 > 2:
          print("Five is greater than two!")
                print("This will cause an error!")

Python Variables

In Python, variables are created when you assign a value to them. Unlike some programming languages that require variable declaration, Python dynamically creates variables as you assign values:

x = 5
        y = "Hello, World!"

In the above example, x is assigned the integer value 5, and y is assigned the string value "Hello, World!". Python allows you to assign values of different types to variables without explicit type declarations.

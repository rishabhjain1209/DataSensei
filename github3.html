Lesson 3.1: Introduction to GitHub

GitHub is a web-based platform that provides Git repository hosting along with a range of collaboration features. It allows teams to work on projects, manage code repositories, and track issues and enhancements.

Lesson 3.2: Creating a GitHub Account

To get started with GitHub, you need to create an account:

  1. Visit GitHub's website.
  2. Click on "Sign up" and follow the registration process.

Lesson 3.3: Creating a Repository on GitHub

GitHub repositories are used to store and manage your code:

  1. Log in to your GitHub account.
  2. Click the '+' sign in the top-right corner and select "New repository."
  3. Fill in the repository name, description, and other settings.
  4. Click "Create repository."

Lesson 3.4: Cloning Repositories from GitHub

To work on a GitHub repository locally, you need to clone it:

git clone

For example:

git clone https://github.com/username/my-repo.git

Lesson 3.5: Pushing to GitHub (push)

After making changes to your local repository, you can push those changes to GitHub:

git push origin

For example, to push the changes in your "main" branch:

git push origin main

Lesson 3.6: Pulling from GitHub (pull)

To update your local repository with changes from GitHub, use the pull command:

git pull origin

For example, to pull changes from the "main" branch:

git pull origin main

These lessons will help you get started with GitHub, from creating an account and repositories to pushing and pulling code changes. GitHub is a valuable tool for collaboration and version control in software development.

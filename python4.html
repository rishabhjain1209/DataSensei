Python Sets

Sets in Python are a versatile way to store collections of data. They are one of the four built-in data types in Python, with the other three being List, Tuple, and Dictionary. Sets have unique characteristics that make them suitable for specific use cases.

Characteristics of Sets:

  1. Unordered: Sets do not have a defined order for their elements. This means that the items in a set can appear in a different order each time you use them. Unlike lists or tuples, you cannot access set items by index.
  2. Unchangeable (Immutable): Once a set is created, you cannot change its items individually. However, you can add new items and remove existing ones.
  3. No Duplicate Values: Sets do not allow duplicate values. If you attempt to add an element that already exists in the set, it will be ignored.

Creating a Set:

Sets are written with curly braces {}. You can create a set by listing its elements within the braces.

myset = {"apple", "banana", "cherry"}
print(myset)

Output:

{'banana', 'apple', 'cherry'}

As sets are unordered, the order of elements in the set may vary.

Adding and Removing Items:

You can add items to a set using the add() method and remove items using the remove() method. Here's an example:

fruits = {"apple", "banana", "cherry"}
fruits.add("orange")
fruits.remove("banana")
print(fruits)

Output:

{'apple', 'orange', 'cherry'}

Checking Set Length:

To find out how many items are in a set, you can use the len() function:

myset = {"apple", "banana", "cherry"}
print(len(myset))

Output:

3

Data Types in Sets:

Sets can contain items of any data type, including strings, integers, and booleans:

set1 = {"apple", "banana", "cherry"}
set2 = {1, 5, 7, 9, 3}
set3 = {True, False, False}

Sets can even contain different data types within the same set:

mixed_set = {"abc", 34, True, 40, "male"}

Using the set() Constructor:

You can also create a set using the set() constructor:

thisset = set(("apple", "banana", "cherry")) # note the double round-brackets
print(thisset)

Output:

{'banana', 'apple', 'cherry'}

Python Collections (Arrays):

Python offers several collection data types to cater to various requirements:

  • List: Lists are ordered, changeable, and allow duplicate members.
  • Tuple: Tuples are ordered, unchangeable, and allow duplicate members.
  • Set: Sets are unordered, unchangeable, and do not allow duplicate members.
  • Dictionary: Dictionaries are ordered (in Python 3.7+), changeable, and do not allow duplicate keys.

The choice of collection type depends on the specific use case and requirements, such as whether order or duplication matters.

Here's an example illustrating the characteristics and operations of sets:

fruits.add("orange")
fruits.remove("banana")
print(len(fruits))
print(fruits)

Output:

        2
        {'apple', 'orange', 'cherry'}

This example demonstrates the creation, addition, removal, and length calculation of a set in Python.



Access Set Items in Python

In Python, sets are unordered collections of unique elements. You cannot access items in a set by referring to an index or a key like you would with a list or dictionary. However, you can loop through set items using a for loop or check if a specified value is present in a set using the in keyword.

1. Looping Through a Set:

You can use a for loop to iterate through the items in a set and print their values:

thisset = {"apple", "banana", "cherry"}
for x in thisset:
    print(x)

In this code, the for loop iterates through the thisset and prints each item in the set. The result will be:

cherry
banana
apple

Notice that the order of the items in the set may not be the same as the order in which they were added because sets are unordered.

2. Checking for a Specific Value:

You can check if a specific value is present in a set using the in keyword:

thisset = {"apple", "banana", "cherry"}
print("banana" in thisset)

In this code, we check if the value "banana" is present in the thisset. The result will be:

True

The in keyword returns True if the specified value is found in the set; otherwise, it returns False.

Change Items:

Once a set is created, you cannot change its items (sets are mutable), but you can add new items or remove existing ones. The next section covers adding items to sets.

Add Items to a Set:

To add one item to a set, you can use the add() method:

thisset = {"apple", "banana", "cherry"}
thisset.add("orange")
print(thisset)

In this code, we use the add() method to add the item "orange" to the thisset. The result will include "orange" as part of the set:

{'orange', 'banana', 'cherry', 'apple'}

The add() method adds the item to the set if it's not already present. If the item is already in the set, it will not be added again.

Add Sets:

To add items from another set into the current set, you can use the update() method:

thisset = {"apple", "banana", "cherry"}
tropical = {"pineapple", "mango", "papaya"}
thisset.update(tropical)
print(thisset)

In this code, we use the update() method to add elements from the tropical set into the thisset. The result will be a set containing all the items from both sets:

{'apple', 'papaya', 'mango', 'banana', 'pineapple', 'cherry'}

You can add elements from any iterable object (e.g., tuples, lists, dictionaries) using the update() method.

These methods are useful for working with sets and adding or checking for items within them.



Remove Set Items

In Python, you can remove items from a set using various methods, such as remove(), discard(), or pop(). Here's an explanation of these methods:

1. remove() Method:

The remove() method is used to remove a specified item from the set. If the item to remove does not exist, it will raise an error.

thisset = {"apple", "banana", "cherry"}
thisset.remove("banana")
print(thisset)

In this code, "banana" is removed from the set, and the updated set is printed:

{'apple', 'cherry'}

2. discard() Method:

The discard() method is also used to remove a specified item from the set. However, if the item does not exist, discard() will not raise an error.

thisset = {"apple", "banana", "cherry"}
thisset.discard("banana")
print(thisset)

The result will be the same as with the remove() method:

{'apple', 'cherry'}

3. pop() Method:

The pop() method removes and returns a random item from the set. Since sets are unordered collections, you cannot predict which item will be removed. The removed item is returned as the result of the pop() method.

thisset = {"apple", "banana", "cherry"}
x = thisset.pop()
print("Removed item:", x)
print("Updated set:", thisset)

The result will vary each time you run the code, but it may look like this:

Removed item: cherry
Updated set: {'banana', 'apple'}

4. clear() Method:

The clear() method empties the entire set, making it an empty set:

thisset = {"apple", "banana", "cherry"}
thisset.clear()
print(thisset)

The result will be an empty set:

set()

5. Using del Keyword:

You can also use the del keyword to delete the entire set:

thisset = {"apple", "banana", "cherry"}
del thisset

After using del, the set no longer exists.

Loop Through Set Items

Looping through set items in Python allows you to iterate over the elements of a set and perform operations on each element. Here's an explanation of how to loop through set items along with a code example:

Loop Through Set Items:

    print(fruit)

In this code, we define a set called fruits that contains three fruit names. We then use a for loop to iterate through the set, and for each iteration, we print the value of the current item. The output will be:

banana
apple
cherry

The order of items in the output may not necessarily match the order in which they were added to the set because sets are unordered collections.

You can use this looping mechanism to perform various operations on each item in the set, such as checking conditions, modifying values, or performing calculations.



Joining Sets

Joining sets in Python allows you to combine the elements from multiple sets into a single set. Here's an explanation of how to join sets along with code examples:

Join Two Sets:

There are two common methods to join two sets:

1. union() Method:

The union() method returns a new set containing all items from both sets. It creates a new set without modifying the original sets.

# Define two sets
set1 = {"a", "b", "c"}
set2 = {1, 2, 3}
# Use the union() method to combine sets
set3 = set1.union(set2)
print(set3)

Output:

{1, 2, 3, 'a', 'b', 'c'}

In this example, set3 contains all items from both set1 and set2, creating a new set.

2. update() Method:

The update() method inserts all the items from one set into another, effectively modifying the first set. It adds elements from the second set to the first set.

# Define two sets
set1 = {"a", "b", "c"}
set2 = {1, 2, 3}
# Use the update() method to insert set2 into set1
set1.update(set2)
print(set1)

Output:

{1, 2, 3, 'a', 'b', 'c'}

In this example, set1 is modified to include all items from set2.

Both methods result in a new set containing all unique items from both original sets. You can choose the method that best fits your use case, depending on whether you want to create a new set or modify an existing one.



Set Methods in Python

Here are explanations and code examples for various set methods in Python:

1. add():

Adds an element to the set.

my_set = {1, 2, 3}
my_set.add(4)
print(my_set)

Output:

{1, 2, 3, 4}

Removes all the elements from the set.

Returns a copy of the set.

Returns a set containing the difference between two or more sets.

set1 = {1, 2, 3}

Removes the items in this set that are also included in another specified set.

set1 = {1, 2, 3}

{1, 2}

6. discard():

Remove the specified item.

my_set = {1, 2, 3}

print(my_set)

7. intersection():

Returns a set that is the intersection of two other sets.

set1 = {1, 2, 3}
set2 = {3, 4, 5}
intersection_set = set1.intersection(set2)
print(intersection_set)

Output:

{3}

8. intersection_update():

Removes the items in this set that are not present in other specified sets.

set1 = {1, 2, 3}
set2 = {3, 4, 5}
set1.intersection_update(set2)

{3}

Returns whether two sets have an intersection or not.

set1 = {1, 2, 3}
set2 = {4, 5, 6}
is_disjoint = set1.isdisjoint(set2)

True

10. issubset():

Returns whether another set contains this set or not.

set1 = {1, 2}

print(is_subset)

Output:

True

These are some of the essential set methods in Python. You can use these methods to manipulate and work with sets effectively.



Dictionary in Python

Dictionaries are used to store data values in key:value pairs. They are a collection that is ordered (as of Python 3.7), changeable, and does not allow duplicates.

Creating a Dictionary:

Dictionaries are defined using curly braces {} and contain key-value pairs separated by colons :.

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}

Accessing Dictionary Items:

You can access the values in a dictionary by referring to its key.

print(thisdict["brand"])

Ordered vs. Unordered:

In Python 3.7 and later, dictionaries are ordered, meaning that the items have a defined order, and that order will not change. In earlier Python versions (3.6 and earlier), dictionaries are unordered.

Changeable:

Dictionaries are changeable, so you can modify, add, or remove items after creating the dictionary.

Duplicates Not Allowed:

Dictionaries cannot have two items with the same key. If you specify the same key multiple times, the duplicate values will overwrite existing ones.

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964,
  "year": 2020
}
print(thisdict)

Dictionary Length:

To find the number of items in a dictionary, you can use the len() function.

print(len(thisdict))

Dictionary Items - Data Types:

The values in dictionary items can be of any data type, including strings, integers, booleans, and even lists.

thisdict = {
  "brand": "Ford",
  "electric": False,
  "year": 1964,
  "colors": ["red", "white", "blue"]
}

You can use dictionaries to store and organize data efficiently, making it easy to access values using keys.



Type of a Dictionary

In Python, you can use the type() function to check the data type of an object. When you use type() on a dictionary, it returns dict.

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
print(type(thisdict))

Creating a Dictionary Using the dict() Constructor

You can create a dictionary using the dict() constructor by providing key-value pairs or keyword arguments.

thisdict = dict(name="John", age=36, country="Norway")
print(thisdict)

Collections in Python

Python has several collection data types to store and manipulate data efficiently. Here's a brief summary:

  • List: A collection that is ordered and changeable, allowing duplicate members.
    • mylist = [1, 2, 3, 2, 4]
  • Tuple: A collection that is ordered and unchangeable, allowing duplicate members.
    • mytuple = ("apple", "banana", "apple")
  • Set: A collection that is unordered, unchangeable, and does not allow duplicates.
    • myset = {"apple", "banana", "cherry"}
  • Dictionary: A collection that is ordered (Python 3.7+), changeable, and does not allow duplicate keys.
    • mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}

Choosing the Right Collection Type

When selecting a collection type, consider the properties of each type:

  • Lists are suitable when you need an ordered collection that allows duplicates and can change.
  • Tuples are suitable when you need an ordered collection that does not change and allows duplicates.
  • Sets are suitable when you need an unordered collection without duplicates.
  • Dictionaries are suitable for storing key-value pairs, and in Python 3.7+, they maintain the order of insertion.

Choosing the right collection type is essential for efficiency and clarity in your code.



Accessing Dictionary Items

Copying a Dictionary

When copying dictionaries, it's important to understand that simply using dict2 = dict1 will create a reference to the original dictionary, so changes in dict1 will affect dict2. To make an independent copy, you can use the copy() method or the dict() function.

Using the copy() method:

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
mydict = thisdict.copy()

Using the dict() function:

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
mydict = dict(thisdict)

Both of these methods create a new dictionary mydict that is a separate copy of thisdict.

Common Dictionary Methods in Python

1. clear()

Removes all elements from the dictionary.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
mydict.clear()

2. copy()

Returns a copy of the dictionary.

original_dict = {"brand": "Ford", "model": "Mustang", "year": 1964}
copy_dict = original_dict.copy()

3. fromkeys()

Returns a dictionary with the specified keys and a default value.

keys = ["brand", "model", "year"]
default_value = "N/A"
new_dict = dict.fromkeys(keys, default_value)

4. get()

Returns the value of the specified key.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
model = mydict.get("model")

5. items()

Returns a list containing a tuple for each key-value pair.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
item_list = mydict.items()

6. keys()

Returns a list containing the dictionary's keys.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
key_list = mydict.keys()

7. pop()

Removes the element with the specified key.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
mydict.pop("model")

8. popitem()

Removes the last inserted key-value pair.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
mydict.popitem()

9. setdefault()

Returns the value of the specified key. If the key doesn't exist, it inserts the key with the specified value.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
value = mydict.setdefault("color", "red")

10. update()

Updates the dictionary with the specified key-value pairs.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
mydict.update({"color": "red", "price": 25000})

11. values()

Returns a list of all the values in the dictionary.

mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}
value_list = mydict.values()

These methods are useful for manipulating dictionaries in various ways, such as adding, removing, or updating key-value pairs, or extracting keys and values for further processing.

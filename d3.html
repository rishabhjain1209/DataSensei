Week 3: Scales and Axes

Introduction to Scales:

In data visualization, scales are essential tools for mapping data values to visual properties such as position, size, and color. Scales help transform raw data into values that can be used to create meaningful visual representations. D3.js provides several types of scales to accommodate different types of data and visual encodings.

Linear Scales:

Linear scales are used to map a continuous input domain to an output range. They are suitable for numeric data that has a linear relationship. Here's how you can create a linear scale in D3.js:

const scale = d3.scaleLinear()
    .domain([dataMin, dataMax]) // Input data domain
    .range([outputMin, outputMax]); // Output range
  • domain([dataMin, dataMax]): Specifies the range of input data values.
  • range([outputMin, outputMax]): Defines the range of output values.

Ordinal Scales:

Ordinal scales are used for categorical or discrete data. They map discrete input values to discrete output values, typically for positioning elements on an axis. Here's an example:

const scale = d3.scaleOrdinal()
    .domain(["Category A", "Category B", "Category C"]) // Input data domain
    .range([10, 50, 90]); // Output range
  • domain(["Category A", "Category B", "Category C"]): Specifies the input categories.
  • range([10, 50, 90]): Defines the output positions for each category.

Color Scales:

Color scales are a type of ordinal scale used to map data values to colors. They are often used for thematic maps or heatmaps. D3.js provides color scales like d3.scaleOrdinal() for this purpose.

Creating Axes with D3:

In data visualization, axes are graphical elements used to indicate the scale and values of a chart. D3.js makes it easy to create axes for your visualizations.

HTML File (index.html):

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Scales and Axes Example</title>
    <script src="https://d3js.org/d3.v6.min.js"></script>
</head>
<body>
    <h1>Scales and Axes Example</h1>
    <svg width="400" height="300" id="visualization"></svg>
    <script>
        // Sample data
        const data = [10, 20, 30, 40, 50];
        // Create a linear scale
        const xScale = d3.scaleLinear()
            .domain([0, d3.max(data)]) // Input data domain
            .range([0, 300]); // Output range
        // Create an axis generator
        const xAxis = d3.axisBottom(xScale);
        // Create an SVG container
        const svg = d3.select("#visualization");
        // Append the x-axis to the SVG
        svg.append("g")
            .attr("class", "x-axis")
            .attr("transform", "translate(50, 200)") // Position the axis
            .call(xAxis); // Call the axis generator
    </script>
</body>
</html>

In this example:

  • We define a sample data array [10, 20, 30, 40, 50].
  • We create a linear scale xScale to map data values to positions.
  • We use the d3.axisBottom(xScale) function to create a bottom-oriented axis generator.
  • We create an SVG container and append the x-axis to it.

When you open this HTML file in your web browser, you'll see a horizontal axis at the bottom of the SVG, indicating the scale of the data values. This demonstrates how to create a basic axis using D3.js along with scales for data transformation.

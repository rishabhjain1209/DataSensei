Module 4: Data Visualization

Creating 2D and 3D Plots:

MATLAB offers a wide range of functions and tools for creating both 2D and 3D plots to visualize data. Some commonly used plotting functions include:

  1. plot: Creates line plots to display data points connected by lines.
  2. scatter: Generates scatter plots to visualize individual data points.
  3. bar and barh: Create vertical and horizontal bar charts.
  4. hist: Produces histograms to display the distribution of data.
  5. pie: Creates pie charts to represent data as slices of a circle.

3D Plots:

  • plot3: Generates 3D line plots.
  • scatter3: Creates 3D scatter plots.
  • surf: Generates 3D surface plots.
  • contour: Produces 2D contour plots of 3D data.
  • slice: Creates 3D slice plots to visualize data in planes.

Customizing Plot Appearance:

MATLAB provides extensive options for customizing the appearance of your plots. You can modify properties like line styles, colors, markers, labels, axes, and more. Common customization tasks include:

  • Setting Line Styles and Colors: - Use '-', '--', ':', and '-.' to specify line styles. - Use RGB values or predefined color names (e.g., 'r' for red) to set line colors.
  • Adding Markers: - Specify markers (e.g., 'o', 's', 'x') to highlight data points.
  • Adjusting Axes: - Customize axis labels and limits using xlabel, ylabel, zlabel, xlim, ylim, and zlim.
  • Adding Legends: - Use legend to add legends that describe data series.
  • Title and Labels: - Add titles and labels to your plots using title, xlabel, and ylabel.
  • Grids and Ticks: - Control grid lines and tick marks with grid, xticks, yticks, and zticks.
  • Annotations and Text: - Add text annotations to your plots using text or annotation.

Annotating Plots with Labels and Titles:

You can annotate your plots with labels, titles, and other text using MATLAB's text and annotation functions. For example:

x = 1:10;
y = rand(1, 10);
plot(x, y, 'o-'); % Plot data
xlabel('X-axis'); % Label the X-axis
ylabel('Y-axis'); % Label the Y-axis
title('Scatter Plot Example'); % Add a title
text(3, 0.7, 'Data Point 3'); % Add text annotation at a specific location
legend('Data Series'); % Add a legend

Generating Statistical Charts and Graphs:

MATLAB provides functions for generating various statistical charts and graphs to analyze data distributions, relationships, and trends. Some common statistical charts include:

  1. Histograms: You can use the hist function to create histograms to visualize data distribution.
  2. Box Plots: The boxplot function generates box plots to display the median, quartiles, and potential outliers in data.
  3. Scatter Plots with Regression Lines: You can create scatter plots with regression lines using scatter and lsline functions to visualize linear relationships.
  4. QQ Plots: The qqplot function generates quantile-quantile (QQ) plots to assess data normality.
  5. Heatmaps: MATLAB's heatmap function can be used to create heatmaps for visualizing data with color intensity.
  6. Error Bars: Use errorbar to display error bars on plots, which are common in scientific data representation.

Here's an example of creating a histogram and a box plot in MATLAB:

data = randn(1, 100); % Generate random data
% Create a histogram
histogram(data, 'BinWidth', 0.5);
xlabel('Value');
ylabel('Frequency');
title('Histogram of Random Data');
% Create a box plot
boxplot(data);
xlabel('Data');
ylabel('Value');
title('Box Plot of Random Data');

MATLAB's data visualization capabilities make it a powerful tool for exploring and presenting data in various formats and styles to aid in data analysis and interpretation.

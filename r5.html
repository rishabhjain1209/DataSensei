Conditional Statements in R:

1. `if-else` Statements:

Conditional statements in R allow you to make decisions based on certain conditions. The `if-else` construct allows you to execute different blocks of code depending on whether a condition is `TRUE` or `FALSE`.

Syntax of an `if-else` Statement:


if (condition) {
  # Code to execute if the condition is TRUE
} else {
  # Code to execute if the condition is FALSE
}

Example of an `if-else` Statement:


# Example: Determine if a number is positive or negative
x <- -5
if (x > 0) {
  print("The number is positive.")
} else {
  print("The number is not positive (i.e., zero or negative).")
}

Nested `if-else` Statements:


# Example: Nested if-else statements to determine the grade based on a score
score <- 85
if (score >= 90) {
  grade <- "A"
} else {
  if (score >= 80) {
    grade <- "B"
  } else {
    if (score >= 70) {
      grade <- "C"
    } else {
      grade <- "D"
    }
  }
}
print(paste("Your grade is:", grade))

Vectorized `if-else` with `ifelse()`: 


# Example: Using ifelse() for vectorized conditional operations
scores <- c(75, 88, 92, 60, 78)
pass_fail <- ifelse(scores >= 70, "Pass", "Fail")
print(pass_fail)

`if-else` statements are fundamental for controlling the flow of your R programs, allowing you to make decisions and execute different code paths based on conditions.



Loops in R:

1. `for` Loops:

A `for` loop in R allows you to iterate over a sequence of values and execute a block of code for each value in the sequence.

Syntax of a `for` Loop:


for (variable in sequence) {
  # Code to be executed in each iteration
}

Example of a `for` Loop:


# Example: Print numbers from 1 to 5 using a for loop
for (i in 1:5) {
  print(i)
}

2. `while` Loops:

A `while` loop in R continues to execute a block of code as long as a specified condition is `TRUE`. It is used when you don't know in advance how many times the loop will run.

Syntax of a `while` Loop:


while (condition) {
  # Code to be executed while the condition is TRUE
}

Example of a `while` Loop:


# Example: Print numbers from 1 to 5 using a while loop
i <- 1
while (i <= 5) {
  print(i)
  i <- i + 1
}

Common Looping Patterns:

Loops can be used for various purposes, including iterating through data structures, performing calculations, or executing repetitive tasks. Here are some common looping patterns:

  • Summing elements in a vector.
  • Finding the maximum or minimum value in a vector.
  • Counting occurrences of a specific value in a vector.
  • Iterating through rows or columns of a data frame.
  • Repeating an operation until a convergence criterion is met.

Example: Summing Elements in a Vector using a `for` Loop:


# Example: Sum elements in a vector using a for loop
numbers <- c(1, 2, 3, 4, 5)
sum_result <- 0
for (num in numbers) {
  sum_result <- sum_result + num
}
print(paste("The sum is:", sum_result))

Loops are powerful tools for automating repetitive tasks and processing data iteratively. However, it's important to use them carefully to avoid infinite loops or inefficient code. In many cases, you can achieve the same results more efficiently using vectorized operations and functions in R.

Lesson 5: Creating Bar Charts and Line Charts

  1. Building Bar Charts

    Bar charts are useful for displaying categorical data. Here's how to create one in Tableau:

    • Drag a dimension (e.g., Product Category) to the Columns shelf.
    • Drag a measure (e.g., Sales) to the Rows shelf.
    • Tableau will automatically create a bar chart.

    Example Code for Creating a Bar Chart:

                1. Connect to your data source.
            2. Drag "Product Category" to Columns.
            3. Drag "Sales" to Rows.
            4. Tableau creates a bar chart showing sales by product category.

  2. Constructing Line Charts

    Line charts are effective for visualizing trends over time or continuous data. Here's how to create one:

    • Drag a dimension or date field to the Columns shelf.
    • Drag a measure to the Rows shelf.
    • Change the mark type to "Line."

    Example Code for Creating a Line Chart:

                1. Connect to your data source.
            2. Drag "Date" to Columns.
            3. Drag "Revenue" to Rows.
            4. Change the mark type to "Line."

  3. Adding Filters and Sorting

    To enhance your visualizations, you can add filters to allow users to interact with the data. To add a filter:

    • Drag a dimension (e.g., Region) to the Filters shelf.
    • Users can now select specific regions to filter the data.

    To sort your visualizations:

    • Right-click a dimension or measure.
    • Choose "Sort" and select the desired sorting option (e.g., ascending or descending).

Lesson 6: Pie Charts and Scatter Plots

  1. Creating Pie Charts

    Pie charts are suitable for showing parts of a whole. To create one in Tableau:

    • Drag a dimension (e.g., Product Category) to the "Color" shelf.
    • Drag a measure (e.g., Sales) to the "Angle" shelf.
    • Tableau will create a pie chart.

    Example Code for Creating a Pie Chart:

                1. Connect to your data source.
            2. Drag "Product Category" to Color.
            3. Drag "Sales" to Angle.
            4. Tableau generates a pie chart representing sales by product category.

  2. Building Scatter Plots

    Scatter plots are excellent for visualizing relationships between two continuous variables. To create one:

    • Drag a measure to the Columns shelf.
    • Drag another measure to the Rows shelf.
    • Tableau will create a scatter plot.

    Example Code for Creating a Scatter Plot:

                1. Connect to your data source.
            2. Drag "Profit" to Columns.
            3. Drag "Sales" to Rows.
            4. Tableau generates a scatter plot showing the relationship between profit and sales.

  3. Customizing Visuals

    Tableau provides extensive customization options, such as changing colors, labels, and formatting axes. To customize a visualization:

    • Click on various elements in the chart.
    • Use the "Marks" card to adjust the appearance and behavior of marks.
    • Explore the "Format" menu for further customization options.

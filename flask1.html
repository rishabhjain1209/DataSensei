Lesson 1: Introduction to Web Frameworks and Flask

In Lesson 1, we'll introduce you to the concept of web frameworks and dive into Flask, a micro web framework for Python. Web frameworks are essential tools for building web applications efficiently. Flask is known for its simplicity and flexibility, making it an excellent choice for beginners and experienced developers alike.

What is a Web Framework?

  • A web framework is a pre-built collection of tools, libraries, and best practices that simplifies web development.
  • It provides a structured way to create web applications, handle HTTP requests, and manage routes.

Why Flask?

  • Flask is lightweight, making it easy to get started with.
  • It follows the WSGI (Web Server Gateway Interface) standard, making it compatible with various web servers.
  • Flask allows you to start small and add components as your project grows.

Lesson 2: Setting Up a Development Environment

Before we start building Flask applications, we need to set up a development environment. This includes installing Python, Flask, and any other necessary tools.

Step 1: Install Python

  • Download and install Python from the official website.
  • Ensure that Python is added to your system's PATH during installation.

Step 2: Create a Virtual Environment

  1. Open your terminal or command prompt.
  2. Navigate to your project directory.
  3. Run the following commands to create and activate a virtual environment:
python -m venv venv # Create a virtual environment
source venv/bin/activate # Activate the virtual environment (Linux/macOS)
venv\Scripts\activate # Activate the virtual environment (Windows)

Step 3: Install Flask

  • With your virtual environment activated, install Flask using pip:
pip install Flask

Lesson 3: Your First Flask Application: Hello, World!

Now that your development environment is set up, let's create your first Flask application. We'll create a simple "Hello, World!" web page.

  1. Create a Python file, e.g., app.py, in your project directory.
  2. Open app.py and add the following code:
        
            from flask import Flask
            app = Flask(__name__)
            @app.route('/')
            def hello_world():
                return 'Hello, World!'
            if __name__ == '__main__':
                app.run()

Here's what this code does:

  • We import the Flask class and create a Flask web application instance.
  • We define a route using the @app.route('/') decorator, which tells Flask to execute the hello_world function when someone accesses the root URL ('/').
  • The hello_world function simply returns the string 'Hello, World!'.
  1. Save the file and return to your terminal.
  2. Make sure your virtual environment is activated (as shown in Lesson 2).
  3. Run the Flask application:
python app.py

You'll see output indicating that the Flask development server is running. Open your web browser and go to http://localhost:5000/. You should see "Hello, World!" displayed in your browser.

Congratulations! You've just created and run your first Flask application. In the upcoming lessons, we'll explore more advanced Flask features and build more complex web applications.

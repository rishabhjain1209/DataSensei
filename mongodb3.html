Lesson 9: Inserting Documents

In MongoDB, you can insert documents into collections using various methods, including insertOne() and insertMany().

Example of inserting a single document:

const newUser = {
  name: "Alice",
  age: 28,
  email: "alice@example.com"
};
// Insert the document into the 'users' collection
db.collection('users').insertOne(newUser, (err, result) => {
  if (err) throw err;
  console.log("New user added:", result.insertedId);
});

Example of inserting multiple documents:

const newUsers = [
  { name: "Bob", age: 35, email: "bob@example.com" },
  { name: "Charlie", age: 22, email: "charlie@example.com" }
];
// Insert the documents into the 'users' collection
db.collection('users').insertMany(newUsers, (err, result) => {
  if (err) throw err;
  console.log("New users added:", result.insertedIds);
});

Lesson 10: Querying Documents

MongoDB provides powerful querying capabilities to retrieve documents from collections. You can use methods like find() and various query operators to filter and sort results.

Example of querying documents:

// Find all users with age greater than 30
db.collection('users').find({ age: { $gt: 30 } }).toArray((err, users) => {
  if (err) throw err;
  console.log("Users over 30:", users);
});

Lesson 11: Updating Documents

Updating documents in MongoDB can be done using the updateOne() and updateMany() methods. You can modify specific fields or replace entire documents.

Example of updating a single document:

// Update the email of the user with name "Alice"
db.collection('users').updateOne(
  { name: "Alice" },
  { $set: { email: "newalice@example.com" } },
  (err, result) => {
    if (err) throw err;
    console.log("User updated:", result.modifiedCount);
  }
);

Example of updating multiple documents:

// Increase the age of all users by 1
db.collection('users').updateMany(
  {},
  { $inc: { age: 1 } },
  (err, result) => {
    if (err) throw err;
    console.log("Users updated:", result.modifiedCount);
  }
);

Lesson 12: Deleting Documents

To remove documents from a collection, you can use the deleteOne() and deleteMany() methods.

Example of deleting a single document:

// Delete the user with name "Charlie"
db.collection('users').deleteOne({ name: "Charlie" }, (err, result) => {
  if (err) throw err;
  console.log("User deleted:", result.deletedCount);
});

Example of deleting multiple documents:

// Delete all users with age less than 25
db.collection('users').deleteMany({ age: { $lt: 25 } }, (err, result) => {
  if (err) throw err;
  console.log("Users deleted:", result.deletedCount);
});

These CRUD operations are fundamental for working with MongoDB and are used extensively in building applications that rely on MongoDB as their data store.

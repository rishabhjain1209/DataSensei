Week 4: Advanced Functions and Formulas in Excel

Topic 1: IF, SUMIF, COUNTIF Functions

IF Function: The IF function allows you to perform conditional calculations. It returns one value if a condition is true and another value if it's false.

' Use the IF function to calculate a bonus based on sales
xlWorksheet.Cells(1, 1).Formula = "=IF(B1>10000, 1000, 500)"

SUMIF Function: The SUMIF function calculates the sum of values in a range that meet a specific condition.

' Use the SUMIF function to calculate the total sales of a specific product
xlWorksheet.Cells(2, 1).Formula = "=SUMIF(A1:A10, 'Product A', B1:B10)"

COUNTIF Function: The COUNTIF function counts the number of cells in a range that meet a specific condition.

' Use the COUNTIF function to count the number of employees with sales > 5000
xlWorksheet.Cells(3, 1).Formula = "=COUNTIF(B1:B10, '>5000')"

Topic 2: VLOOKUP and HLOOKUP Functions

VLOOKUP Function: The VLOOKUP function searches for a value in the first column of a table and returns a corresponding value from a specified column.

' Use the VLOOKUP function to find the price of a product based on its code
xlWorksheet.Cells(4, 1).Formula = "=VLOOKUP('Product Code', A1:B10, 2, FALSE)"

HLOOKUP Function: The HLOOKUP function is similar to VLOOKUP but searches for a value in the first row of a table and returns a corresponding value from a specified row.

' Use the HLOOKUP function to find the total sales for a specific month
xlWorksheet.Cells(5, 1).Formula = "=HLOOKUP('Jan', A1:D2, 2, FALSE)"

Topic 3: Text Functions (CONCATENATE, LEFT, RIGHT, MID)

CONCATENATE Function: The CONCATENATE function combines multiple text strings into one.

' Use the CONCATENATE function to combine first name and last name
xlWorksheet.Cells(6, 1).Formula = "=CONCATENATE('John', ' ', 'Doe')"

LEFT Function: The LEFT function extracts a specified number of characters from the beginning of a text string.

' Use the LEFT function to get the first 3 characters of a text
xlWorksheet.Cells(7, 1).Formula = "=LEFT('Hello, World!', 3)"

RIGHT Function: The RIGHT function extracts a specified number of characters from the end of a text string.

' Use the RIGHT function to get the last 5 characters of a text
xlWorksheet.Cells(8, 1).Formula = "=RIGHT('Excel is fun', 5)"

MID Function: The MID function extracts a specified number of characters from a text string, starting at a specified position.

' Use the MID function to extract "Excel" from the text
xlWorksheet.Cells(9, 1).Formula = "=MID('Microsoft Excel', 11, 5)"

Topic 4: Date and Time Functions

Excel offers various date and time functions to work with date and time values. Here's an example of using the TODAY and NOW functions:

' Use the TODAY function to display the current date
xlWorksheet.Cells(10, 1).Formula = "=TODAY()"
' Use the NOW function to display the current date and time
xlWorksheet.Cells(11, 1).Formula = "=NOW()"

Topic 5: Logical Functions (AND, OR, NOT)

Logical functions help evaluate logical conditions in Excel. Here's an example of using the AND, OR, and NOT functions:

' Use the AND function to check if both conditions are true
xlWorksheet.Cells(12, 1).Formula = "=AND(B1>10, C1<20)"
' Use the OR function to check if at least one condition is true
xlWorksheet.Cells(13, 1).Formula = "=OR(D1>30, E1<5)"
' Use the NOT function to reverse the result of a condition
xlWorksheet.Cells(14, 1).Formula = "=NOT(F1=0)"

These code examples cover the use of advanced functions and formulas in Excel, including IF, SUMIF, COUNTIF, VLOOKUP, HLOOKUP, text functions (CONCATENATE, LEFT, RIGHT, MID), date and time functions (TODAY, NOW), and logical functions (AND, OR, NOT). These functions are essential for performing complex calculations and data manipulations in Excel.

Python Lists

Lists in Python are used to store multiple items in a single variable. Lists are a fundamental data type and are one of the four built-in data types in Python used to store collections of data, the other three being Tuple, Set, and Dictionary. Lists are versatile and can store items of different data types.

Creating Lists

Lists are created using square brackets []. Here's an example:


thislist = ["apple", "banana", "cherry"]
print(thislist)

List Items

  • Ordered: List items have a defined order that will not change. If you add new items to a list, they will be placed at the end of the list.
  • Changeable: Lists are mutable, meaning you can change, add, and remove items after creating the list.
  • Allow Duplicates: Lists can have items with the same value.

Example of Duplicate Values


thislist = ["apple", "banana", "cherry", "apple", "cherry"]
print(thislist)

List Length

To find out how many items a list contains, you can use the len() function:


thislist = ["apple", "banana", "cherry"]
print(len(thislist))

List Items - Data Types

List items can be of any data type, including strings, integers, and booleans. Lists can also contain a mix of different data types:


list1 = ["apple", "banana", "cherry"]
list2 = [1, 5, 7, 9, 3]
list3 = [True, False, False]
list4 = ["abc", 34, True, 40, "male"]

Checking Data Type

You can check the data type of a list using the type() function:


mylist = ["apple", "banana", "cherry"]
print(type(mylist))

Using the list() Constructor

You can also create a new list using the list() constructor. This constructor accepts an iterable (e.g., a tuple or another list) as an argument and creates a new list from it:


thislist = list(("apple", "banana", "cherry"))  
print(thislist)

This is especially useful when you want to convert other iterable types into lists.



Python Collections (Arrays)

Python provides four main collection data types to store multiple items: Lists, Tuples, Sets, and Dictionaries. Each collection type has its own characteristics, making it suitable for different use cases.

List:

  • Ordered and changeable: Lists maintain the order of elements, and you can modify their contents after creation.
  • Allows duplicate members: You can have multiple items with the same value.

Example:


mylist = ["apple", "banana", "cherry"]

Tuple:

  • Ordered and unchangeable: Tuples maintain order, but their elements cannot be modified after creation.
  • Allows duplicate members: Like lists, tuples can also contain duplicate items.

Example:


mytuple = ("apple", "banana", "cherry")

Set:

  • Unordered and unchangeable (though items can be added or removed): Sets do not have a defined order, and you cannot modify individual items.
  • No duplicate members: Sets automatically remove duplicate values.

Example:


myset = {"apple", "banana", "cherry"}

Dictionary:

  • Ordered (as of Python 3.7) and changeable: Dictionaries maintain order and allow modifications.
  • No duplicate members: Dictionaries store key-value pairs, and keys must be unique.

Example:


mydict = {"brand": "Ford", "model": "Mustang", "year": 1964}

Range of Negative Indexes:

You can use negative indexes to start searching from the end of a list. For example, to get items from "orange" (-4) to, but not including, "mango" (-1):


thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"]
print(thislist[-4:-1])

Check if Item Exists:

You can determine if a specified item is present in a list using the in keyword:


thislist = ["apple", "banana", "cherry"]
if "apple" in thislist:
  print("Yes, 'apple' is in the fruits list")

Changing List Items:

Change Item Value: You can change the value of a specific item in a list by referring to its index:


thislist = ["apple", "banana", "cherry"]
thislist[1] = "blackcurrant"

Change a Range of Item Values: To change the values of items within a specific range, define a list with the new values and refer to the range of index numbers where you want to insert the new values:


thislist = ["apple", "banana", "cherry", "orange", "kiwi", "mango"]
thislist[1:3] = ["blackcurrant", "watermelon"]

Insert Items: To insert a new list item without replacing any existing values, you can use the insert() method. The method inserts an item at the specified index:


thislist = ["apple", "banana", "cherry"]
thislist.insert(2, "watermelon")

These collection data types provide various ways to store and manipulate data, making Python a versatile programming language for various tasks.



Adding Items to a List in Python

In Python, you can add items to a list using several methods. Here are the main methods for adding items to a list:

  1. Append Items:

    To add an item to the end of a list, you can use the append() method. It adds the specified item as the last element in the list.

    
thislist = ["apple", "banana", "cherry"]
thislist.append("orange")

    This will result in thislist containing ["apple", "banana", "cherry", "orange"].

  2. Insert Items:

    You can insert an item at a specific index using the insert() method. It allows you to specify the position at which you want to insert the new item.

    
thislist = ["apple", "banana", "cherry"]
thislist.insert(1, "orange")

    This will result in thislist containing ["apple", "orange", "banana", "cherry"].

  3. Extend List:

    To append elements from another list to the current list, you can use the extend() method. It adds the elements from another list to the end of the current list.

    
thislist = ["apple", "banana", "cherry"]
tropical = ["mango", "pineapple", "papaya"]
thislist.extend(tropical)

    This will result in thislist containing ["apple", "banana", "cherry", "mango", "pineapple", "papaya"].

  4. Add Any Iterable:

    The extend() method can also add elements from other iterable objects like tuples, sets, and dictionaries. It doesn't have to be limited to lists.

    
thislist = ["apple", "banana", "cherry"]
thistuple = ("kiwi", "orange")
thislist.extend(thistuple)

    This will result in thislist containing ["apple", "banana", "cherry", "kiwi", "orange"].

Adding items to a list is a fundamental operation in Python when working with collections of data. Depending on your specific needs, you can choose the method that best suits your use case.



Looping Through Lists in Python

In Python, you can loop through lists using various methods. Here are some common techniques:

  1. Loop Through a List with a for Loop:

    You can use a for loop to iterate through the items in a list one by one. Here's an example:

    
thislist = ["apple", "banana", "cherry"]
for x in thislist:
    print(x)

    This code will print each item in the list on a separate line.

  2. Loop Through the Index Numbers:

    If you want to loop through the list items by referring to their index numbers, you can use the range() and len() functions to create an iterable. Here's an example:

    
thislist = ["apple", "banana", "cherry"]
for i in range(len(thislist)):
    print(thislist[i])

    This code creates an iterable [0, 1, 2] and prints each item using its index.

  3. Using a while Loop:

    Another way to loop through a list is by using a while loop. You can determine the length of the list with len() and then loop through the list items by referring to their indexes. Don't forget to increment the index after each iteration. Here's an example:

    
thislist = ["apple", "banana", "cherry"]
i = 0
while i < len(thislist):
    print(thislist[i])
    i = i + 1

    This code achieves the same result as the previous example but uses a while loop.

  4. Looping Using List Comprehension:

    List comprehension provides a concise way to loop through lists and perform operations on each item. You can use list comprehension to create a new list or execute a specific action for each item. Here's an example:

    
thislist = ["apple", "banana", "cherry"]
[print(x) for x in thislist]

    This code uses list comprehension to print each item in the list.

Looping through lists is a fundamental skill when working with collections of data in Python. Depending on your specific use case, you can choose the looping method that best suits your needs.



List Comprehension in Python

List comprehension is a concise way to create new lists based on the values of an existing list. It allows you to filter and manipulate the elements of a list in a single line of code.

  1. Filtering a List Without List Comprehension:

    Suppose you have a list of fruits and you want to create a new list containing only the fruits with the letter "a" in their name. Without list comprehension, you would typically use a for loop with a conditional statement and an empty list to achieve this:

    fruits = ["apple", "banana", "cherry", "kiwi", "mango"]
newlist = []
for x in fruits:
    if "a" in x:
        newlist.append(x)
print(newlist)

    This code will print ['apple', 'banana', 'mango'].

  2. Filtering a List Using List Comprehension:

    With list comprehension, you can achieve the same result with a more compact code:

    fruits = ["apple", "banana", "cherry", "kiwi", "mango"]
newlist = [x for x in fruits if "a" in x]
print(newlist)

    This code produces the same output, ['apple', 'banana', 'mango'], in just one line.

Copying Lists in Python

Copying lists in Python requires careful consideration because simply using list2 = list1 will create a reference to the same list rather than a new copy. Here are two common methods to create copies of lists:

  1. Using the copy() Method:

    You can use the copy() method to create a shallow copy of a list:

    thislist = ["apple", "banana", "cherry"]
mylist = thislist.copy()

    Both thislist and mylist will contain the same elements, but they will be two separate lists in memory.

  2. Using the list() Constructor:

    Another way to make a copy is to use the list() constructor:

    thislist = ["apple", "banana", "cherry"]
mylist = list(thislist)

    This method achieves the same result as using copy(), creating an independent copy of the original list.

Python Tuples

Tuples in Python are another way to store collections of data, similar to lists. However, there are some key differences:

  1. Ordered

    Tuples, like lists, are ordered. This means that the items in a tuple have a specific order, and that order will not change.

  2. Unchangeable (Immutable)

    Unlike lists, tuples are unchangeable, which means you cannot add, remove, or change items after creating a tuple. Once you define a tuple, it remains the same throughout its lifetime.

  3. Allow Duplicates

    Tuples, like lists, can contain duplicate values.

Creating a Tuple:

Tuples are created by enclosing values in round brackets:

thistuple = ("apple", "banana", "cherry")

Tuple Items:

Tuple items are indexed, starting from 0 for the first item. They are ordered and allow duplicates. Once created, you cannot change the items.

Tuple Length:

To find out how many items a tuple contains, you can use the len() function:

thistuple = ("apple", "banana", "cherry")
print(len(thistuple))  # Output: 3

Creating a Tuple with One Item:

If you want to create a tuple with only one item, you need to include a comma after the item, as without the comma, Python will not recognize it as a tuple:

thistuple = ("apple",)  # One item tuple

Tuple Items and Data Types:

Tuple items can be of any data type, and a tuple can contain items of mixed data types:

tuple1 = ("apple", "banana", "cherry")
tuple2 = (1, 5, 7, 9, 3)
tuple3 = (True, False, False)
tuple4 = ("abc", 34, True, 40, "male")

Type of a Tuple:

To check the data type of a tuple, you can use the type() function:

mytuple = ("apple", "banana", "cherry")
print(type(mytuple))  # Output: <class 'tuple'>

Using the tuple() Constructor:

You can also create a tuple using the tuple() constructor:

thistuple = tuple(("apple", "banana", "cherry"))

Tuples are commonly used when you want to store a collection of values that should not be modified, such as coordinates or elements of a date. They can also be used as keys in dictionaries because of their immutability.



Python Collections - Accessing Tuple Items

In Python, you can access tuple items by referring to their index number. Tuples are ordered, so each item has a specific index, starting from 0. Here's more detail on how to access tuple items:

Accessing by Index:

You can access a tuple item by referring to its index within square brackets. For example, if you have a tuple thistuple, you can access its second item like this:

thistuple = ("apple", "banana", "cherry")
print(thistuple[1])  # Output: banana

Note that the index starts from 0, so the first item is at index 0, the second item at index 1, and so on.

Negative Indexing:

Negative indexing allows you to access items from the end of the tuple. -1 refers to the last item, -2 refers to the second last item, and so on. Here's an example:

thistuple = ("apple", "banana", "cherry")
print(thistuple[-1])  # Output: cherry (last item)

Range of Indexes:

You can specify a range of indexes to access multiple items at once. The range is defined by specifying where to start and where to end the range. The result will be a new tuple containing the specified items.

thistuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
print(thistuple[2:5])  # Output: ('cherry', 'orange', 'kiwi')

In this example, we access items from index 2 (included) to index 5 (not included).

Omitting Start or End Value:

If you omit the start value, the range will start from the beginning of the tuple. If you omit the end value, the range will go to the end of the tuple. Examples:

# Start at the beginning, but exclude "kiwi"
print(thistuple[:4])  # Output: ('apple', 'banana', 'cherry', 'orange')
# Start from "cherry" to the end
print(thistuple[2:])  # Output: ('cherry', 'orange', 'kiwi', 'melon', 'mango')

Range of Negative Indexes:

You can also use negative indexes to specify a range, similar to positive indexes. For example:

# From index -4 (included) to index -1 (excluded)
print(thistuple[-4:-1])  # Output: ('orange', 'kiwi', 'melon')

Checking If an Item Exists:

To determine if a specified item exists in a tuple, you can use the in keyword:

if "apple" in thistuple:
    print("Yes, 'apple' is in the tuple")

This code will check if "apple" exists in thistuple and print a message accordingly.

Tuples are useful when you want to store data that should not be changed, such as a set of coordinates or configuration settings for a program.



Python - Updating Tuples

Tuples are immutable in Python, meaning once they are created, you cannot change their values, add new items, or remove existing ones. However, there are workarounds to achieve these operations.

Changing Tuple Values:

While you can't change a tuple directly, you can convert it to a list, make the changes, and then convert it back to a tuple. Here's an example:

x = ("apple", "banana", "cherry")
y = list(x)
y[1] = "kiwi"
x = tuple(y)
print(x)  # Output: ('apple', 'kiwi', 'cherry')

In this example, we first convert the tuple x into a list y, change the second item to "kiwi," and then convert it back into a tuple.

Adding Items to a Tuple:

Since tuples are immutable, they don't have built-in methods like append to add items. However, you can use the following methods:

1. Convert to List:

Convert the tuple to a list, add the item(s), and then convert it back into a tuple.

thistuple = ("apple", "banana", "cherry")
y = list(thistuple)
y.append("orange")
thistuple = tuple(y)
print(thistuple)  # Output: ('apple', 'banana', 'cherry', 'orange')

2. Add Tuple to Tuple:

You can add one or more tuples to an existing tuple.

thistuple = ("apple", "banana", "cherry")
y = ("orange",)
thistuple += y
print(thistuple)  # Output: ('apple', 'banana', 'cherry', 'orange')

Removing Items from a Tuple:

Tuples are immutable, so you cannot remove items from them directly. However, you can use the following workarounds:

1. Convert to List:

Convert the tuple to a list, remove the item(s), and then convert it back into a tuple.

thistuple = ("apple", "banana", "cherry")
y = list(thistuple)
y.remove("apple")
thistuple = tuple(y)
print(thistuple)  # Output: ('banana', 'cherry')

2. Delete the Tuple:

You can completely delete the tuple using the del keyword.

thistuple = ("apple", "banana", "cherry")
del thistuple
# This will raise an error: NameError: name 'thistuple' is not defined

These workarounds allow you to modify and manipulate tuples indirectly, even though tuples themselves remain immutable.



Python - Unpacking Tuples

In Python, you can both pack and unpack tuples. Packing a tuple means assigning multiple values to a single tuple variable, while unpacking means extracting values from a tuple and assigning them to individual variables.

Unpacking a Tuple:

Unpacking allows you to assign values from a tuple to separate variables. To unpack a tuple, you provide the variable names separated by commas within parentheses, and then assign the tuple you want to unpack on the right side of the assignment.

Example:

fruits = ("apple", "banana", "cherry")
(green, yellow, red) = fruits
print(green)  # Output: apple
print(yellow) # Output: banana
print(red)    # Output: cherry

In this example, we have unpacked the fruits tuple into three variables: green, yellow, and red. Each variable now holds one of the values from the tuple.

Using Asterisk (*) for Unpacking:

If the number of variables you have is less than the number of values in the tuple, you can use an asterisk * to collect the remaining values into a list. This is useful when you have a variable number of items to unpack.

Example:

fruits = ("apple", "banana", "cherry", "strawberry", "raspberry")
(green, yellow, *red) = fruits
print(green)  # Output: apple
print(yellow) # Output: banana
print(red)    # Output: ['cherry', 'strawberry', 'raspberry']

In this case, green and yellow are assigned the first two values, and the remaining values are collected into the red variable as a list.

Unpacking with an Asterisk in the Middle:

You can also use the asterisk * in the middle of the variable names to collect values from the middle of the tuple.

Example:

fruits = ("apple", "mango", "papaya", "pineapple", "cherry")
(green, *tropic, red) = fruits
print(green)   # Output: apple
print(tropic)  # Output: ['mango', 'papaya', 'pineapple']
print(red)     # Output: cherry

In this example, green gets the first value, red gets the last value, and tropic collects the values in between as a list.

Unpacking tuples is a convenient way to access and work with the individual elements of a tuple.



Python - Looping Through Tuples

In Python, you can loop through the items of a tuple using various methods, including for loops and while loops. Here, we'll explore how to loop through tuples and provide code examples for each approach.

Loop Through a Tuple Using a for Loop:

You can use a for loop to iterate through the items in a tuple. Here's an example:

thistuple = ("apple", "banana", "cherry")
for x in thistuple:
  print(x)

This code will output:

apple
banana
cherry

You can see that the loop iterates through the thistuple and prints each item.

Loop Through a Tuple Using the range() and len() Functions:

Another way to loop through a tuple is by referring to the index numbers of its items. You can use the range() and len() functions to create an iterable that represents the indexes. Here's an example:

thistuple = ("apple", "banana", "cherry")
for i in range(len(thistuple)):
  print(thistuple[i])

This code produces the same output as the previous example:

apple
banana
cherry

In this case, we use the range(len(thistuple)) to create an iterable [0, 1, 2], which corresponds to the indexes of the tuple.

Loop Through a Tuple Using a While Loop:

You can also loop through a tuple using a while loop. To do this, use the len() function to determine the length of the tuple and start at index 0. Remember to increment the index by 1 in each iteration. Here's an example:

thistuple = ("apple", "banana", "cherry")
i = 0
while i < len(thistuple):
  print(thistuple[i])
  i = i + 1

This code, too, will produce the same output:

apple
banana
cherry

In this while loop, we start with i = 0, and in each iteration, we print the item at index i and then increment i by 1 until we reach the end of the tuple.

Looping through tuples allows you to process and work with the individual items of the tuple in your Python programs.



Python - Joining and Multiplying Tuples

In Python, you can combine, or join, two or more tuples to create a new tuple. You can also multiply the content of a tuple by a specified number of times. Below, we'll elaborate on both operations and provide code examples.

Join Two Tuples:

To join, or concatenate, two or more tuples, you can use the + operator. Here's an example:

tuple1 = ("a", "b", "c")
tuple2 = (1, 2, 3)
tuple3 = tuple1 + tuple2
print(tuple3)

In this code, tuple1 and tuple2 are combined into tuple3 using the + operator. The output will be:

('a', 'b', 'c', 1, 2, 3)

The resulting tuple, tuple3, contains all the elements from both tuple1 and tuple2.

Multiply Tuples:

If you want to duplicate the content of a tuple a specific number of times, you can use the * operator. Here's an example:

fruits = ("apple", "banana", "cherry")
mytuple = fruits * 2
print(mytuple)

In this code, the fruits tuple is duplicated twice using the * operator. The output will be:

('apple', 'banana', 'cherry', 'apple', 'banana', 'cherry')

As you can see, the content of the fruits tuple is repeated two times in the mytuple.

Elaboration:

Joining tuples using the + operator allows you to merge multiple tuples into a single tuple, while multiplying a tuple with * duplicates its content. These operations can be useful when you need to manipulate tuple data in your Python programs, whether for combining data or repeating a sequence of values.

mytuple = (1, 2, 3, 2, 4, 2, 5)

3

index():

mytuple = ("apple", "banana", "cherry", "banana")
index_banana = mytuple.index("banana")
print(index_banana)

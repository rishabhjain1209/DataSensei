Python Classes and Objects

Python is an object-oriented programming language, which means that it revolves around the concept of objects. Everything in Python is an object, and each object has properties (attributes) and methods (functions) associated with it. Classes are used to define the blueprint for creating objects.

1. Create a Class

To create a class, use the class keyword. Classes are like blueprints for creating objects, defining their attributes and behaviors.

class MyClass:
    x = 5

2. Create an Object

Once a class is defined, you can create objects (instances) of that class. Objects are instances of a class.

p1 = MyClass()
print(p1.x)

3. The __init__() Function

The __init__() function is a special method that gets called when an object is created from a class. It is used to initialize object properties or perform other setup operations.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
p1 = Person("John", 36)
print(p1.name)
print(p1.age)

4. The __str__() Function

The __str__() function is used to control what should be returned when the object is represented as a string. It's a built-in method that you can define for custom string representation of objects.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def __str__(self):
        return f"{self.name} ({self.age})"
p1 = Person("John", 36)
print(p1)

5. Object Methods

Objects can contain methods, which are functions that belong to the object and can access its properties.

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    def myfunc(self):
        print("Hello, my name is " + self.name)
p1 = Person("John", 36)
p1.myfunc()

These are the fundamental concepts of Python classes and objects. Classes define the structure of objects, and objects are instances of classes with properties and methods.



Python Classes and Objects - Advanced Topics

1. The self Parameter

In Python, the self parameter is used as a reference to the current instance of the class. It is the first parameter of any function (method) defined within the class, and it is automatically passed when you call the method on an object. Although self is the convention, you can use any name you like for this parameter.

class Person:
    def __init__(mysillyobject, name, age):
        mysillyobject.name = name
        mysillyobject.age = age
    def myfunc(abc):
        print("Hello, my name is " + abc.name)
p1 = Person("John", 36)
p1.myfunc()

2. Modify Object Properties

You can modify object properties (attributes) by directly accessing and assigning new values to them.

p1.age = 40

3. Delete Object Properties

You can delete object properties (attributes) using the del keyword followed by the object and the property you want to delete.

del p1.age

4. Delete Objects

You can delete objects by using the del keyword followed by the object's name.

del p1

5. The pass Statement

Class definitions cannot be empty, but if you need to create a class with no content temporarily, you can use the pass statement to avoid getting an error.

class Person:
    pass

These advanced topics provide a deeper understanding of working with Python classes and objects, including the use of the self parameter, modifying properties, deleting properties and objects, and the pass statement.



Python Inheritance - Object-Oriented Programming (OOP)

1. Create a Parent Class

In Python, you can create a parent class like this:

class Person:
    def __init__(self, fname, lname):
        self.firstname = fname
        self.lastname = lname
    def printname(self):
        print(self.firstname, self.lastname)

2. Create a Child Class

To create a child class that inherits from the Person class, you can do this:

class Student(Person):
    pass

3. Add the __init__() Function to Child Class

To customize the child class, you can add an __init__() function to it. This function is automatically called when you create an object of the child class. Here's how you add it:

class Student(Person):
    def __init__(self, fname, lname):
        Person.__init__(self, fname, lname)

4. Use the super() Function

Python also provides the super() function to make it more concise:

class Student(Person):
    def __init__(self, fname, lname):
        super().__init__(fname, lname)

5. Add Properties

You can add properties to the child class like this:

class Student(Person):
    def __init__(self, fname, lname, year):
        super().__init__(fname, lname)
        self.graduationyear = year

6. Add Methods

To add methods to the child class, define them within the class like this:

class Student(Person):
    def __init__(self, fname, lname, year):
        super().__init__(fname, lname)
        self.graduationyear = year
    def welcome(self):
        print("Welcome", self.firstname, self.lastname, "to the class of", self.graduationyear)

Now you have a child class Student that inherits from the parent class Person, and you can customize it with additional properties and methods as needed.



Python Inheritance - Using super(), Adding Properties, and Adding Methods

1. Using the super() Function

In Python, the super() function is used to call methods and access properties from the parent class within the child class. It simplifies the process of inheriting methods and properties.

class Student(Person):
    def __init__(self, fname, lname):
        super().__init__(fname, lname)

2. Adding Properties

You can add properties to the child class by defining them within the class. In this example, we add a graduationyear property to the Student class:

class Student(Person):
    def __init__(self, fname, lname, year):
        super().__init__(fname, lname)
        self.graduationyear = year

3. Adding Methods

To add methods to the child class, define them within the class. In this example, we add a welcome() method to the Student class:

class Student(Person):
    def __init__(self, fname, lname, year):
        super().__init__(fname, lname)
        self.graduationyear = year
    def welcome(self):
        print("Welcome", self.firstname, self.lastname, "to the class of", self.graduationyear)

Now, when you create a Student object and call the welcome() method, it will print a welcome message including the student's name and graduation year.

Keep in mind that if you add a method in the child class with the same name as a function in the parent class, the child class's method will override the parent class's method. This allows you to customize the behavior of inherited methods in the child class while still having access to the parent class's methods and properties using super().



Python Iterators

1. What is an Iterator?

An iterator is an object that contains a countable number of values and allows you to traverse through all the values it holds. In Python, an iterator is an object that implements the iterator protocol, consisting of the methods __iter__() and __next__().

2. Iterator vs. Iterable

Iterable objects are objects that can be looped over or iterated upon. Examples of iterable objects in Python include lists, tuples, dictionaries, and sets. To obtain an iterator from an iterable, you can use the iter() method.

mytuple = ("apple", "banana", "cherry")
myit = iter(mytuple)
print(next(myit))
print(next(myit))
print(next(myit))

This code creates an iterator myit from a tuple and uses the next() function to retrieve each value.

3. Iterating Through an Iterator

You can use a for loop to iterate through an iterable object, which internally creates an iterator object and uses the next() method for iteration.

mytuple = ("apple", "banana", "cherry")
for x in mytuple:
  print(x)

In this example, we use a for loop to iterate through the values of the tuple mytuple without explicitly creating an iterator.

Additionally, here's an example of iterating through characters in a string:

mystr = "banana"
for x in mystr:
  print(x)

This loop iterates through each character in the string mystr.



Polymorphism in Python

1. Function Polymorphism:

In Python, you can achieve function polymorphism with built-in functions that can be used on different types of objects. One example is the len() function, which behaves differently depending on the type of object it's applied to.

x = "Hello World!"
print(len(x))  # Outputs: 12
mytuple = ("apple", "banana", "cherry")
print(len(mytuple))  # Outputs: 3
thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
print(len(thisdict))  # Outputs: 3

2. Class Polymorphism:

Polymorphism is often used in class methods where multiple classes share the same method name but provide their own implementation. For example, you can have different classes (Car, Boat, Plane) with a common method move().

class Car:
  def move(self):
    print("Drive!")
class Boat:
  def move(self):
    print("Sail!")
class Plane:
  def move(self):
    print("Fly!")
car1 = Car()
boat1 = Boat()
plane1 = Plane()
for vehicle in (car1, boat1, plane1):
    vehicle.move()

3. Inheritance and Class Polymorphism:

You can also achieve polymorphism in classes through inheritance. Child classes can inherit methods from a parent class and provide their own implementation if needed.

class Vehicle:
  def move(self):
    print("Move!")
class Car(Vehicle):
  pass
class Boat(Vehicle):
  def move(self):
    print("Sail!")
class Plane(Vehicle):
  def move(self):
    print("Fly!")
car1 = Car()
boat1 = Boat()
plane1 = Plane()
for vehicle in (car1, boat1, plane1):
    vehicle.move()


Python Scope

1. Local Scope:

A variable created inside a function belongs to the local scope of that function, and it can only be accessed within that function. It is not visible outside the function.

def myfunc():
  x = 300
  print(x)
myfunc()
# Outputs: 300

2. Function Inside Function:

Variables defined in an outer function are also accessible within inner functions. This is known as nested scope.

def myfunc():
  x = 300
  def myinnerfunc():
    print(x)
  myinnerfunc()
myfunc()
# Outputs: 300

3. Global Scope:

A variable created in the main body of Python code (outside of any function) is a global variable. It belongs to the global scope, which means it can be accessed from any part of the code.

x = 300
def myfunc():
  print(x)
myfunc()
# Outputs: 300
print(x)
# Outputs: 300

4. Naming Variables:

If you use the same variable name both inside and outside a function, Python treats them as two separate variables. One is in the global scope, and the other is in the local scope of the function.

x = 300
def myfunc():
  x = 200
  print(x)
myfunc()
# Outputs: 200
print(x)
# Outputs: 300

5. Global Keyword:

To create or modify a global variable from within a function, you can use the global keyword.

def myfunc():
  global x
  x = 300
myfunc()
print(x)
# Outputs: 300
    
x = 300
def myfunc():
  global x
  x = 200
myfunc()
print(x)
# Outputs: 200


Python Modules

1. Create a Module:

To create a module, save your code in a file with the .py file extension. This file can contain functions, variables, classes, or any Python code.

        Example: mymodule.py
        ```python
def greeting(name):
    print("Hello, " + name)
person1 = {
    "name": "John",
    "age": 36,
    "country": "Norway"
}

2. Use a Module:

You can use a module in your code by using the import statement. To access functions, variables, or other elements in the module, use the syntax: module_name.element_name.

        Example: Using the mymodule Module
        ```python
import mymodule
mymodule.greeting("Jonathan")
a = mymodule.person1["age"]
print(a)

3. Variables in a Module:

Modules can contain variables of all types, including dictionaries, lists, and objects. These variables can be accessed after importing the module.

        Example: Accessing a Dictionary in mymodule
        ```python
import mymodule
a = mymodule.person1["age"]
print(a)

4. Naming a Module:

You can name the module file whatever you like, but it must have the .py file extension.

5. Re-naming a Module:

You can create an alias for a module when importing it using the as keyword. This allows you to use a shorter name for the module in your code.

        Example: Creating an Alias for mymodule
        ```python
import mymodule as mx
a = mx.person1["age"]
print(a)

6. Built-in Modules:

Python comes with several built-in modules that you can import and use in your programs. These modules provide various functionalities, such as working with files, handling dates and times, and more.

        Example: Using the platform Module
        ```python
import platform
x = platform.system()
print(x)

7. Using the dir() Function:

The dir() function allows you to list all the defined names (functions, variables, classes, etc.) in a module. This is helpful for exploring the contents of a module.

        Example: Listing Names in the platform Module
        ```python
import platform
x = dir(platform)
print(x)

8. Importing Specific Elements from a Module:

You can import only specific elements from a module using the from keyword. This way, you don't need to use the module name when referring to those elements in your code.

        Example: Importing person1 from mymodule
        ```python
from mymodule import person1
print(person1["age"])


Python Datetime

1. Importing the datetime Module:

You can import the datetime module to work with date and time-related operations.

        Example: Importing datetime and Displaying the Current Date and Time
        ```python
import datetime
x = datetime.datetime.now()
print(x)

2. Date Output:

A datetime object contains information about the year, month, day, hour, minute, second, and microsecond. You can access these components using various methods.

        Example: Accessing Year and Weekday Name
        ```python
import datetime
x = datetime.datetime.now()
print(x.year)
print(x.strftime("%A"))

3. Creating Date Objects:

You can create date objects using the datetime class constructor. This class requires three parameters: year, month, and day. You can also specify time-related parameters like hour, minute, second, and microsecond, but they are optional.

        Example: Creating a Date Object
        ```python
import datetime
x = datetime.datetime(2020, 5, 17)
print(x)

4. The strftime() Method:

The strftime() method of the datetime object allows you to format date and time objects into readable strings. You pass a format string as a parameter to specify how the string should be formatted.

        Example: Formatting Month Name
        ```python
import datetime
x = datetime.datetime(2018, 6, 1)
print(x.strftime("%B"))

Here are some commonly used format codes for strftime():

        %d: Day of the month (e.g., "31").
        %b: Month name, short version (e.g., "Dec").
        %B: Month name, full version (e.g., "December").
        %m: Month as a number (e.g., "12").
        %y: Year, short version without century (e.g., "18").
        %Y: Year, full version (e.g., "2018").
        %H: Hour (00-23, e.g., "17").
        %I: Hour (00-12, e.g., "05").
        %M: Minute (00-59, e.g., "41").
        %S: Second (00-59, e.g., "08").
        %f: Microsecond (e.g., "548513").
        %z: UTC offset (e.g., "+0100").
        %Z: Timezone (e.g., "CST").
        %j: Day number of the year (001-366, e.g., "365").
        %U: Week number of the year (Sunday as the first day of the week, 00-53, e.g., "52").
        %W: Week number of the year (Monday as the first day of the week, 00-53, e.g., "52").
        %c: Local version of date and time (e.g., "Mon Dec 31 17:41:00 2018").
        %x: Local version of date (e.g., "12/31/18").
        %X: Local version of time (e.g., "17:41:00").
        %%: A % character.
        %G: ISO 8601 year (e.g., "2018").
        %u: ISO 8601 weekday (1-7, e.g., "1").
        %V: ISO 8601 week number (01-53, e.g., "01").


Python Math

Python provides built-in mathematical functions and a math module that extends the capabilities for performing various mathematical tasks on numbers.

1. Built-in Math Functions:

min() and max()

These functions help find the lowest or highest value in an iterable.

x = min(5, 10, 25)
y = max(5, 10, 25)

abs()

The abs() function returns the absolute (positive) value of the specified number.

x = abs(-7.25)

pow(x, y)

The pow() function returns the value of x to the power of y (xy).

x = pow(4, 3)

Python JSON (JavaScript Object Notation)

JSON is a text-based format for storing and exchanging data, widely used for data interchange between a server and a web application, or between different parts of an application. Python provides a built-in package called json for working with JSON data.

1. Importing the json Module:

Example: Importing the json Module

import json

2. Parsing JSON (Converting from JSON to Python):

Example: Converting from JSON to Python

import json
# JSON string
x = '{ "name": "John", "age": 30, "city": "New York" }'
# Parse x
y = json.loads(x)
# Result is a Python dictionary
print(y["age"])  # Output: 30

3. Converting from Python to JSON:

Example: Converting from Python to JSON

import json
# Python dictionary
x = {
  "name": "John",
  "age": 30,
  "city": "New York"
}
# Convert to JSON
y = json.dumps(x)
# Result is a JSON string
print(y)

4. Supported Python Data Types:

Example: Converting Python Objects into JSON

import json
print(json.dumps({"name": "John", "age": 30}))
print(json.dumps(["apple", "bananas"]))
print(json.dumps(("apple", "bananas")))
print(json.dumps("hello"))
print(json.dumps(42))
print(json.dumps(31.76))
print(json.dumps(True))
print(json.dumps(False))
print(json.dumps(None))

5. Complex Python Object to JSON:

Example: Converting a Complex Python Object

import json
x = {
  "name": "John",
  "age": 30,
  "married": True,
  "divorced": False,
  "children": ("Ann", "Billy"),
  "pets": None,
  "cars": [
    {"model": "BMW 230", "mpg": 27.5},
    {"model": "Ford Edge", "mpg": 24.1}
  ]
}
print(json.dumps(x))

6. Formatting the Result:

Example: Formatting the JSON Result

import json
x = {
  "name": "John",
  "age": 30,
  "city": "New York"
}
# Format with indentation
print(json.dumps(x, indent=4))
# Format with custom separators
print(json.dumps(x, indent=4, separators=(". ", " = ")))

7. Ordering the Result:

Example: Ordering the JSON Result

import json
x = {
  "name": "John",
  "age": 30,
  "city": "New York"
}
# Sort keys in the result
print(json.dumps(x, indent=4, sort_keys=True))

JSON is a versatile format for data storage and exchange, and the json module in Python makes it easy to work with JSON data in your programs.

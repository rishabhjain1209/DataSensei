Lesson 9: Creating and Handling HTML Forms in Flask

In Lesson 9, we'll explore how to create HTML forms in Flask and handle user input. Forms are crucial for collecting data from users, such as text input, checkboxes, and radio buttons.

Creating HTML Forms:

  • In HTML, use the <form> element to define a form.
  • Include input fields (e.g., text, password, radio buttons) and a submit button.
  • Assign a unique name attribute to each input element.

Example of a basic HTML form (form.html):

        
            <!DOCTYPE html>
            <html>
            <head>
                <title>Contact Us</title>
            </head>
            <body>
                <h1>Contact Us</h1>
                <form method="POST" action="/submit">
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name" required><br>
                    <label for="email">Email:</label>
                    <input type="email" id="email" name="email" required><br>                  
                    <label for="message">Message:</label>
                    <textarea id="message" name="message" required></textarea><br>                    
                    <input type="submit" value="Submit">
                </form>
            </body>
            </html>

Lab: Building a Form-Based Application (e.g., a Contact Form)

In the lab, you'll build a form-based application, like a contact form, using Flask. Here's a step-by-step guide:

  1. Create an HTML form (e.g., form.html) with input fields for user data.
  2. Define a route in your Flask application (e.g., /contact) that handles both GET and POST requests.
  3. In the route, use request.method to differentiate between GET (displaying the form) and POST (handling form submission) requests.
  4. Retrieve and validate form data from request.form for POST requests.
  5. Implement the desired actions based on the form data, such as sending an email or saving it to a database.
  6. Return a response to the user, acknowledging the submission.

By completing this lab, you'll gain practical experience in creating forms, handling user input, and building form-based applications in Flask.

File Handling in Python:

File handling is essential for working with data in Python. Python provides the open() function for working with files, and there are different modes to specify how you want to interact with a file.

Modes for Opening Files:

  1. "r" (Read): This mode is used for reading the content of a file. It's the default mode, and it raises an error if the file doesn't exist.
  2. "a" (Append): This mode is used for appending data to a file. If the file doesn't exist, it creates a new one.
  3. "w" (Write): This mode is used for writing data to a file. If the file doesn't exist, it creates a new one. If it exists, it truncates the existing content.
  4. "x" (Create): This mode is used to create a new file. It returns an error if the file already exists.

Binary and Text Modes:

You can also specify whether the file should be handled as binary or text mode:

  • "t" (Text): This is the default value and is used for text mode.
  • "b" (Binary): This mode is used for binary files like images.

Opening a File:

To open a file, use the open() function. For example, to open a file for reading:

f = open("demofile.txt", "r")

If the file is located in a different directory, specify the full file path:

f = open("D:\\myfiles\\welcome.txt", "r")

Reading a File:

# Open a file for reading
f = open("demofile.txt", "r")
# Read and print the content of the file
print(f.read())
# Close the file when you're done
f.close()

Reading a Specific Number of Characters:

# Open a file for reading
f = open("demofile.txt", "r")
# Read and print the first 5 characters of the file
print(f.read(5))
# Close the file when you're done
f.close()

Reading Lines from a File:

# Open a file for reading
f = open("demofile.txt", "r")
# Read and print one line from the file
print(f.readline())
# Close the file when you're done
f.close()

Looping Through Lines of a File:

# Open a file for reading
f = open("demofile.txt", "r")
# Loop through and print each line of the file
for line in f:
    print(line)
# Close the file when you're done
f.close()

These examples cover the basics of file handling in Python, including opening, reading, and looping through files. Remember to close the file after you're done working with it to free up system resources.



File Operations in Python:

Working with files is an essential part of many Python applications. Let's explore various file operations and provide code examples for each topic:

Closing Files:

It's essential to close files after you've finished using them to free up system resources and ensure any changes you made are saved.

# Open a file for reading
f = open("demofile.txt", "r")
# Read and print one line from the file
print(f.readline())
# Close the file when you're done
f.close()

Writing to an Existing File:

You can write to an existing file using modes like "a" (append) or "w" (write). Append mode appends content to the end of the file, while write mode overwrites the existing content.

# Append content to an existing file
f = open("demofile2.txt", "a")
f.write("Now the file has more content!")
f.close()
# Open and read the file after appending
f = open("demofile2.txt", "r")
print(f.read())

Overwriting an Existing File:

If you want to overwrite the entire content of an existing file, use write mode "w".

# Overwrite content of an existing file
f = open("demofile3.txt", "w")
f.write("Woops! I have deleted the content!")
f.close()
# Open and read the file after overwriting
f = open("demofile3.txt", "r")
print(f.read())

Creating a New File:

You can create a new file using "x" mode, which returns an error if the file already exists, or "w" mode, which creates a new file if it doesn't exist.

# Create a new file (raises an error if it exists)
f = open("myfile.txt", "x")
# Create a new file (or overwrite if it exists)
f = open("myfile.txt", "w")

Deleting a File:

To delete a file in Python, you can use the os.remove() function from the os module.

import os
# Remove the file "demofile.txt"
os.remove("demofile.txt")

Checking if a File Exists Before Deleting:

It's a good practice to check if a file exists before attempting to delete it to avoid errors.

import os
# Check if file exists, then delete it
if os.path.exists("demofile.txt"):
    os.remove("demofile.txt")
else:
    print("The file does not exist")

Deleting a Folder:

To delete an entire folder, use the os.rmdir() method from the os module. Note that this method can only remove empty folders.

import os
# Remove the folder "myfolder"
os.rmdir("myfolder")

These examples cover various file operations in Python, including opening, reading, writing, and deleting files, as well as deleting folders. Remember to close files and handle exceptions when necessary to ensure your file operations are reliable and error-free.

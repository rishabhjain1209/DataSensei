Module 6: Time Series Data with Pandas

6.1 Working with Date and Time Data

Time series data often involves date and time information, and Pandas provides powerful tools to work with such data.

Code Example: Creating a time series DataFrame and extracting date components.


import pandas as pd
# Creating a sample time series DataFrame
data = {'Date': ['2023-09-01', '2023-09-02', '2023-09-03'],
        'Temperature (C)': [25.5, 26.2, 24.8]}
df = pd.DataFrame(data)
# Converting the 'Date' column to datetime
df['Date'] = pd.to_datetime(df['Date'])
# Extracting date components
df['Year'] = df['Date'].dt.year
df['Month'] = df['Date'].dt.month
df['Day'] = df['Date'].dt.day
print(df)

6.2 Resampling and Rolling Windows

Resampling involves changing the frequency of time series data, e.g., aggregating daily data to monthly. Rolling windows allow you to compute moving statistics.

Code Example: Resampling and calculating a rolling mean.


# Creating a sample time series DataFrame
data = {'Date': pd.date_range(start='2023-01-01', periods=365),
        'Value': range(1, 366)}
df = pd.DataFrame(data)
# Resampling to monthly averages
monthly_avg = df.resample('M', on='Date').mean()
print(monthly_avg.head())
# Calculating a rolling mean over a window of size 7
rolling_mean = df['Value'].rolling(window=7).mean()
print(rolling_mean.head(10))

6.3 Time Series Visualization

Visualizing time series data helps in understanding trends and patterns over time.

Code Example: Creating a time series plot.


import matplotlib.pyplot as plt
# Plotting a time series
df.set_index('Date', inplace=True)
df['Value'].plot(figsize=(10, 5))
plt.title('Time Series Data')
plt.xlabel('Date')
plt.ylabel('Value')
plt.show()

These examples cover essential time series data manipulation and visualization techniques using Pandas. Working with date and time data, resampling, and creating time series plots are critical skills for analyzing and understanding time-based data.

Module 5: Data Visualization with Pandas

5.1 Basic Plotting with Pandas

Pandas provides built-in methods for basic data visualization, allowing you to create simple plots.

Code Example: Creating a basic line plot.

import pandas as pd
import matplotlib.pyplot as plt
# Creating a sample DataFrame
data = {'Year': [2010, 2011, 2012, 2013, 2014],
        'Revenue': [1000, 1200, 1500, 1800, 2000]}
df = pd.DataFrame(data)
# Creating a line plot
df.plot(x='Year', y='Revenue', kind='line', title='Revenue Over Years')
plt.xlabel('Year')
plt.ylabel('Revenue')
plt.show()

5.2 Customizing Plots

You can customize various aspects of your plots, such as labels, colors, and styles.

Code Example: Customizing a line plot.

# Customizing the line plot
df.plot(x='Year', y='Revenue', kind='line', title='Revenue Over Years', color='green', linestyle='--', marker='o')
plt.xlabel('Year')
plt.ylabel('Revenue')
plt.legend(['Revenue'])
plt.grid(True)
plt.show()

5.3 Exploratory Data Visualization

Exploratory data visualization helps you understand data distribution and relationships between variables.

Code Example: Creating a scatter plot for exploratory analysis.

# Creating a sample DataFrame with two variables
data = {'Age': [25, 30, 35, 40, 45],
        'Income': [50000, 60000, 75000, 90000, 80000]}
df = pd.DataFrame(data)
# Creating a scatter plot to explore the relationship between Age and Income
df.plot(x='Age', y='Income', kind='scatter', title='Age vs. Income')
plt.xlabel('Age')
plt.ylabel('Income')
plt.show()

These examples cover fundamental data visualization topics with Pandas, including basic plotting, customization of plots, and exploratory data visualization. Customizing your plots allows you to create visually appealing and informative visuals for your data analysis tasks. Exploratory visualization helps you gain insights into your data and identify patterns or relationships between variables.

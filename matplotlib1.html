Lesson 1: What is Matplotlib?

Overview of Matplotlib's history and purpose:

Matplotlib is a popular Python library used for data visualization. It was originally created by John D. Hunter in 2003 and has since become a powerful tool for creating a wide range of static, animated, and interactive plots and charts. Matplotlib's primary purpose is to help users visualize their data effectively by providing a comprehensive set of tools for creating various types of plots and customizing them to suit their needs.

Installation and setup:

To use Matplotlib, you need to install it first. You can install it using pip:

pip install matplotlib

Once installed, you can import Matplotlib in your Python scripts or Jupyter notebooks.

import matplotlib.pyplot as plt

Lesson 2: Getting Started with Matplotlib

Introduction to the pyplot module:

Matplotlib's pyplot module provides a simple interface for creating and customizing plots. It's the most commonly used module for creating basic plots. Here's a basic example of creating a simple plot using pyplot:

import matplotlib.pyplot as plt
# Sample data
x = [1, 2, 3, 4, 5]
y = [10, 12, 5, 8, 9]
# Create a line plot
plt.plot(x, y)
# Show the plot
plt.show()

Understanding the Figure and Axes objects:

In Matplotlib, a plot is typically composed of two main objects: the Figure and the Axes. The Figure is the top-level container that holds one or more Axes objects, which represent individual plots or subplots within the figure. Here's how you can create a Figure and Axes:

import matplotlib.pyplot as plt
# Create a Figure and a single Axes
fig, ax = plt.subplots()
# Create a line plot on the Axes
ax.plot(x, y)
# Show the plot
plt.show()

Lesson 3: Basic Plot Types

Line plots:

Line plots are used to visualize data points connected by lines. Here's an example of creating a line plot:

import matplotlib.pyplot as plt
# Sample data
x = [1, 2, 3, 4, 5]
y = [10, 12, 5, 8, 9]
# Create a line plot
plt.plot(x, y)
# Add labels and title
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.title('Line Plot Example')
# Show the plot
plt.show()

Scatter plots:

Scatter plots are used to visualize individual data points. Here's an example:

import matplotlib.pyplot as plt
# Sample data
x = [1, 2, 3, 4, 5]
y = [10, 12, 5, 8, 9]
# Create a scatter plot
plt.scatter(x, y)
# Add labels and title
plt.xlabel('X-axis')
plt.ylabel('Y-axis')
plt.title('Scatter Plot Example')
# Show the plot
plt.show()

Bar plots:

Bar plots are used to display data as bars, where the length or height of each bar represents the data value. Here's an example:

import matplotlib.pyplot as plt
# Sample data
categories = ['Category A', 'Category B', 'Category C', 'Category D']
values = [15, 7, 10, 12]
# Create a bar plot
plt.bar(categories, values)
# Add labels and title
plt.xlabel('Categories')
plt.ylabel('Values')
plt.title('Bar Plot Example')
# Show the plot
plt.show()

Histograms:

Histograms are used to visualize the distribution of a dataset. Here's an example:

import matplotlib.pyplot as plt
# Sample data
data = [10, 15, 10, 25, 30, 15, 5, 20, 15, 10]
# Create a histogram
plt.hist(data, bins=5, edgecolor='black')
# Add labels and title
plt.xlabel('Value')
plt.ylabel('Frequency')
plt.title('Histogram Example')
# Show the plot
plt.show()

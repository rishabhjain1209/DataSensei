Week 4: Creating Basic Shapes

Drawing Circles, Rectangles, and Lines:

In D3.js, you can create basic shapes like circles, rectangles, and lines to build visual elements for your data visualizations. Here's how you can draw these shapes:

Circles:

const svg = d3.select("svg");
svg.append("circle")
    .attr("cx", 50)
    .attr("cy", 50)
    .attr("r", 30)
    .style("fill", "blue");

Rectangles:

const svg = d3.select("svg");
svg.append("rect")
    .attr("x", 20)
    .attr("y", 20)
    .attr("width", 60)
    .attr("height", 40)
    .style("fill", "red");

Lines:

const svg = d3.select("svg");
svg.append("line")
    .attr("x1", 10)
    .attr("y1", 10)
    .attr("x2", 90)
    .attr("y2", 90)
    .style("stroke", "green")
    .style("stroke-width", 2);

Adding Interactivity to Basic Shapes:

D3.js allows you to add interactivity to your visualizations by responding to user events like clicks or mouseovers. Here's an example of adding interactivity to a circle:

const svg = d3.select("svg");
const circle = svg.append("circle")
    .attr("cx", 50)
    .attr("cy", 50)
    .attr("r", 30)
    .style("fill", "blue");
circle.on("mouseover", () => {
    circle.style("fill", "red");
});
circle.on("mouseout", () => {
    circle.style("fill", "blue");
});

Grouping and Transforming Elements:

D3.js allows you to group elements together using <g> elements in SVG. This is useful for applying transformations or styling to a group of elements as a whole.

Here's an example of grouping and transforming elements:

const svg = d3.select("svg");
// Create a group element
const group = svg.append("g");
// Add circles to the group
group.append("circle")
    .attr("cx", 50)
    .attr("cy", 50)
    .attr("r", 30)
    .style("fill", "blue");
group.append("circle")
    .attr("cx", 100)
    .attr("cy", 50)
    .attr("r", 30)
    .style("fill", "red");
// Translate the group to move all circles
group.attr("transform", "translate(20, 20)");

In this example, we create a group <g> element and add two circles to it. Then, we use the attr("transform", "translate(20, 20)") method to move the entire group of circles by 20 units in both the x and y directions.

This illustrates how you can create, group, and transform elements in D3.js to build complex visualizations.

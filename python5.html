Python If...Else Statements

1. Basic if Statement

The if statement is used to execute a block of code only if a specific condition is true.

a = 33
b = 200
if b > a:
    print("b is greater than a")

Output: "b is greater than a"

2. elif Statement

The elif keyword is used to check an additional condition if the previous condition is false.

a = 33
b = 33
if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")

Output: "a and b are equal"

3. else Statement

The else keyword is used to execute a block of code when the if and elif conditions are false.

a = 200
b = 33
if b > a:
    print("b is greater than a")
elif a == b:
    print("a and b are equal")
else:
    print("a is greater than b")

Output: "a is greater than b"

4. One-line if Statement

If you have only one statement to execute, you can put it on the same line as the if statement.

a = 10
b = 5
if a > b: print("a is greater than b")

Output: "a is greater than b"

5. One-line if...else Statement

You can also use a one-liner for if...else statements.

a = 2
b = 330
print("A") if a > b else print("B")

Output: "B" (because a is not greater than b)

6. Logical Operators

Logical operators (and, or, not) are used to combine or reverse conditional statements.

a = 200
b = 33
c = 500
if a > b and c > a:
    print("Both conditions are True")
elif a > b or a > c:
    print("At least one of the conditions is True")
if not a > b:
    print("a is NOT greater than b")

Output: "Both conditions are True" and "a is NOT greater than b"

7. Nested if Statements

You can have if statements inside other if statements, creating nested conditions.

x = 41
if x > 10:
    print("Above ten,")
    if x > 20:
        print("and also above 20!")
    else:
        print("but not above 20.")

Output:
"Above ten,"
"and also above 20!"

8. The pass Statement

If an if statement must have some content but you don't want to execute anything, you can use the pass statement.

a = 33
b = 200
if b > a:
    pass

The pass statement doesn't do anything and is used as a placeholder.



Python While Loops

1. Basic while Loop

The while loop is used to repeatedly execute a set of statements as long as a given condition is true.

i = 1
while i < 6:
    print(i)
    i += 1

Output:
1
2
3
4
5

2. The break Statement

The break statement is used to exit the loop prematurely, even if the loop condition is still true.

i = 1
while i < 6:
    print(i)
    if i == 3:
        break
    i += 1

Output:
1
2
3

3. The continue Statement

The continue statement is used to skip the rest of the current iteration and continue with the next iteration of the loop.

i = 0
while i < 6:
    i += 1
    if i == 3:
        continue
    print(i)

Output:
1
2
4
5
6

4. The else Statement

The else statement in a while loop is executed when the loop condition becomes false.

i = 1
while i < 6:
    print(i)
    i += 1
else:
    print("i is no longer less than 6")

Output:
1
2
3
4
5
i is no longer less than 6



Python For Loops

1. Basic for Loop

A for loop in Python is used to iterate over a sequence.

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)

Output:
apple
banana
cherry

2. Looping Through a String

Even strings can be iterated using a for loop.

for x in "banana":
    print(x)

Output:
b
a
n
a
n
a

3. The break Statement

The break statement allows you to exit the loop prematurely.

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    if x == "banana":
        break
    print(x)

Output:
apple

4. The continue Statement

The continue statement allows you to skip the rest of the current iteration.

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    if x == "banana":
        continue
    print(x)

Output:
apple
cherry

5. The range() Function

The range() function generates a sequence of numbers.

for x in range(6):
    print(x)

Output:
0
1
2
3
4
5

6. else in For Loop

The else block in a for loop is executed once after the loop completes its iterations.

for x in range(6):
    print(x)
else:
    print("Finally finished!")

Output:
0
1
2
3
4
5
Finally finished!

7. Nested Loops

You can have nested loops, where one loop is inside another.

adj = ["red", "big", "tasty"]
fruits = ["apple", "banana", "cherry"]
for x in adj:
    for y in fruits:
        print(x, y)

Output:
red apple
red banana
red cherry
big apple
big banana
big cherry
tasty apple
tasty banana
tasty cherry

8. The pass Statement

The pass statement serves as a placeholder when there's no content in the loop.

for x in [0, 1, 2]:
    pass

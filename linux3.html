Module 3: Basic Command-Line Usage

Lesson 1: The Terminal Emulator

In this lesson, we'll introduce the terminal emulator, which provides a text-based interface to interact with a Linux system. It's a powerful tool for performing tasks efficiently and gaining more control over your system.

Terminal Emulator:

  • A terminal emulator is a program that allows you to run a shell, such as Bash or Zsh, to execute commands.
  • Common terminal emulators include GNOME Terminal, Konsole, iTerm (for macOS), and Windows Terminal (for Windows).
  • You can open a terminal emulator by searching for it in your system's applications or using keyboard shortcuts like Ctrl+Alt+T.

No specific code example is needed for this lesson since it's an introduction to the terminal emulator.

Lesson 2: Basic Commands (ls, cd, pwd, mkdir, rmdir)

In this lesson, we'll cover fundamental commands for navigating and managing directories and files in the command line.

Basic Commands:

  • ls: List the contents of a directory.
  • cd: Change the current directory.
  • pwd: Print the current working directory.
  • mkdir: Create a new directory.
  • rmdir: Remove an empty directory.

Code Example (Listing Contents of a Directory and Changing Directory):

ls
cd myfolder

These commands list the contents of the current directory and then change to the myfolder directory.

Lesson 3: File and Directory Operations (cp, mv, rm)

In this lesson, we'll explore essential commands for copying, moving, and deleting files and directories in the command line.

File and Directory Operations:

  • cp: Copy files or directories from one location to another.
  • mv: Move or rename files and directories.
  • rm: Remove files and directories.

Code Example (Copying and Moving Files):

cp file.txt backup/
mv oldname.txt newname.txt

These commands copy file.txt to the backup directory and then rename oldname.txt to newname.txt.

Code Example (Removing Files and Directories):

rm unwanted.txt
rm -r myfolder

The first command removes the file unwanted.txt, and the second command removes the myfolder directory and its contents. Be cautious when using rm -r as it deletes directories and their contents recursively.

Lesson 4: File Viewing and Editing (cat, less, nano, vim)

In this lesson, we'll explore commands for viewing and editing text files in the command line, allowing you to read and modify their contents.

File Viewing and Editing Commands:

  • cat: Display the entire content of a file.
  • less: View and navigate through large text files.
  • nano: A simple text editor for editing files in the terminal.
  • vim: A powerful text editor with advanced features.

Code Example (Viewing a File with cat and less):

cat myfile.txt
less largefile.log

The first command displays the entire content of myfile.txt, while the second command opens largefile.log in a pager that allows you to navigate through it.

Code Example (Editing a File with nano and vim):

nano newfile.txt
vim important.txt

These commands open newfile.txt in the nano text editor and important.txt in the vim text editor, enabling you to edit their contents.

By mastering these basic command-line operations and text file handling, you'll become more proficient in using the terminal for various tasks. Practice and experimentation are key to building your command-line skills.

Lesson 1.1: What is Version Control?

Version control is a system that helps developers track changes to their code and collaborate with others effectively. It allows you to record different versions of your project, revert to previous states, and manage code history. Popular version control systems include Git, SVN, and Mercurial.

Lesson 1.2: Introduction to Git

Git is a distributed version control system widely used in the software development industry. It was created by Linus Torvalds and is known for its speed and flexibility. Git helps teams manage code and track changes efficiently.

Lesson 1.3: Installing Git

To get started with Git, you need to install it on your computer. The installation process varies depending on your operating system.

  • On Linux (Ubuntu), you can use the package manager:
    • sudo apt-get install git
  • On macOS, you can use Homebrew:
    • brew install git
  • On Windows, you can download and install Git from the official website.

Lesson 1.4: Basic Git Configuration

Before using Git, it's essential to configure your username and email address. This information will be associated with your commits.

git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

Lesson 1.5: Creating Your First Repository

Now, let's create your first Git repository. A repository is a container for your project's files and their entire history.

  1. Create a new directory for your project:
    2. mkdir my_project
cd my_project
  2. Initialize a Git repository in this directory:
    3. git init
  3. Create a new file (e.g., index.html) and add some content:
    4. <!DOCTYPE html>
<html>
<head>
    <title>My First Git Project</title>
</head>
<body>
    <h1>Hello, Git!</h1>
</body>
</html>
  4. Add this file to the staging area to prepare it for a commit:
    5. git add index.html
  5. Commit your changes with a descriptive message:
    6. git commit -m "Initial commit: Added index.html"

Congratulations! You've just created your first Git repository, added a file, and made your first commit.

Lesson 6: Introduction to Jinja2 Templating Engine

In Lesson 6, we'll introduce you to Jinja2, a powerful templating engine used in Flask. Jinja2 allows you to create dynamic HTML templates by embedding Python code within your HTML files.

Key Concepts:

  • Jinja2 templates use curly braces {} to denote expressions.
  • You can include control structures like loops and conditionals within templates.
  • Templates help separate the presentation (HTML) from the logic (Python).

Example template (template.html):

        
            <!DOCTYPE html>
            <html>
            <head>
                <title>{{ title }}</title>
            </head>
            <body>
                <h1>Welcome, {{ user }}!</h1>
                <ul>
                    {% for item in items %}
                        <li>{{ item }}</li>
                    {% endfor %}
                </ul>
            </body>
            </html>

Lesson 7: Rendering HTML Templates

In Lesson 7, we'll show you how to render HTML templates in Flask. Rendering templates allows you to generate dynamic web pages by injecting data into the templates.

Rendering Templates in Flask:

        
            from flask import Flask, render_template
            app = Flask(__name__)
            @app.route('/')
            def index():
                title = 'My Website'
                user = 'John Doe'
                items = ['Item 1', 'Item 2', 'Item 3']
                return render_template('template.html', title=title, user=user, items=items)
            if __name__ == '__main__':
                app.run()

In this example:

  • We use the render_template function to render the template.html file.
  • We pass variables like title, user, and items to the template.

Lesson 8: Handling Static Files (CSS, JavaScript, Images)

Lesson 8 covers the management of static files such as CSS, JavaScript, and images in Flask. Static files enhance the presentation and interactivity of your web applications.

Organizing Static Files:

  • Create a folder named static in your project directory.
  • Place your static files (e.g., CSS, JS, images) inside the static folder.

Linking Static Files in Templates:

  • Use the url_for function in templates to generate URLs for static files.

Example of linking a CSS file in a template (template.html):

        
            <!DOCTYPE html>
            <html>
            <head>
                <title>{{ title }}</title>
                <link rel="stylesheet" href="{{ url_for('static', filename='style.css') }}">
            </head>
            <body>
                <!-- Content goes here -->
            </body>
            </html>

Comments in Python

In Python, comments are used for various purposes, such as documenting code, making it more readable, and temporarily disabling code during testing. Comments start with a # symbol, and Python treats everything after # on the same line as a comment. Here's a closer look at how comments are used in Python:

Purpose of Comments

  1. Explain Code: Comments are used to explain and provide documentation for your Python code. They help other developers (and your future self) understand the purpose and functionality of different parts of your code.
  2. Readability: Well-placed comments can enhance the readability of your code by providing context and explanations for complex logic or algorithms.
  3. Testing and Debugging: Comments can be used to temporarily disable a block of code during testing or debugging, allowing you to isolate issues without deleting code.

Creating a Comment

Comments in Python start with a # symbol, and Python ignores everything on the same line after the #. Here's how to create comments:

Example 1: Inline Comment

# This is a comment.
print("Hello, World!")

In this example, the # This is a comment. is an inline comment explaining the purpose of the following code.

Example 2: Comment at the End of a Line

You can also place comments at the end of a line of code. Python will ignore everything after the # on that line.

print("Hello, World!")  # This is a comment

The comment # This is a comment provides additional information about the print statement.

Example 3: Disabling Code

You can use comments to temporarily disable code for testing or debugging purposes:

# This line is commented out, so it won't execute.
# print("Hello, World!")
# This line will execute.
print("Cheers, Mate!")

In this example, the first print statement is commented out, so it won't run, while the second print statement executes.

Multiline Comments

Python does not have a dedicated syntax for multiline comments like some other programming languages. However, you can achieve multiline comments using two methods:

Method 1: Insert # for Each Line

You can insert a # symbol at the beginning of each line to create a multiline effect:

# This is a comment
# written in
# more than just one line
print("Hello, World!")

This method works, but it can be cumbersome for long comments.

Method 2: Multiline String (Docstring)

A more common approach to multiline comments in Python is to use a multiline string (often referred to as a docstring). Python will ignore string literals that are not assigned to a variable. So, you can place your comment inside triple-quotes (''' or """) as follows:

"""
This is a comment
written in
more than just one line
"""
print("Hello, World!")

This approach is cleaner and more readable for multiline comments.

Remember that, while the second method is commonly used for documentation and multiline comments, it does create a string object, which might have implications for memory usage if used excessively.



Python Variables

Variables in Python are containers used to store data values. Here's a more detailed explanation of variables in Python:

Creating Variables

Python does not require explicit variable declaration. A variable is created automatically the moment you assign a value to it. You can create variables with meaningful names to store data. For example:

x = 5
y = "John"
print(x)
print(y)

In this example, x is assigned the integer value 5, and y is assigned the string value "John". Notice that you don't need to specify data types explicitly; Python infers the data type based on the assigned value.

Casting

If you want to specify the data type of a variable, you can use casting. Casting allows you to convert one data type to another. Here are some casting examples:

x = str(3)    # x will be a string '3'
y = int(3)    # y will be an integer 3
z = float(3)  # z will be a float 3.0

Getting the Data Type

You can determine the data type of a variable using the type() function:

x = 5
y = "John"
print(type(x))
print(type(y))

This code will print the data types of x and y.

Single or Double Quotes for Strings

String variables can be declared using either single (') or double (") quotes. Both are valid:

x = "John"
# is the same as
x = 'John'

Case-Sensitivity

Variable names in Python are case-sensitive. This means that variables with different letter casing (e.g., myVar, myvar, and MYVAR) are considered distinct variables.

a = 4
A = "Sally"
# A will not overwrite a

Variable Names

Python variable names must adhere to certain rules:

  • A variable name must start with a letter (a-z, A-Z) or an underscore _.
  • A variable name cannot start with a number (0-9).
  • A variable name can contain letters, numbers, and underscores (a-z, A-Z, 0-9, _).
  • Variable names are case-sensitive (e.g., myVar and myvar are two different variables).
  • Variable names cannot be any of Python's reserved keywords.

Legal Variable Names:

myvar = "John"
my_var = "John"
_my_var = "John"
myVar = "John"
MYVAR = "John"
myvar2 = "John"

Illegal Variable Names:

2myvar = "John"
my-var = "John"
my var = "John"

Multi-Word Variable Names

When creating variable names with multiple words, it's a good practice to make them more readable. There are various conventions for this:

  • Camel Case: Each word, except the first, starts with a capital letter (e.g., myVariableName).
  • Pascal Case: Each word starts with a capital letter (e.g., MyVariableName).
  • Snake Case: Each word is separated by an underscore character (e.g., my_variable_name).

Assigning Multiple Values

Python allows you to assign values to multiple variables in a single line:

Multiple Values to Multiple Variables:

x, y, z = "Orange", "Banana", "Cherry"
print(x)
print(y)
print(z)

One Value to Multiple Variables:

You can also assign the same value to multiple variables in one line:

x = y = z = "Orange"
print(x)
print(y)
print(z)

Outputting Variables

To display variables in Python, you often use the print() function:

x = "Python is awesome"
print(x)

You can output multiple variables in a single print() statement, separated by commas:

x = "Python"
y = "is"
z = "awesome"
print(x, y, z)

Alternatively, you can concatenate variables using the + operator:

x = "Python "
y = "is "
z = "awesome"
print(x + y + z)

However, when combining strings and numbers using +, ensure that the types are compatible; otherwise, Python will raise an error.

Global Variables

Variables created outside of any function, at the top-level of your script, are considered global variables. Global variables can be accessed and modified both inside and outside of functions.

Example: Global Variable Used Inside a Function

x = "awesome"
def myfunc():
  print("Python is " + x)
myfunc()

In this example, the x variable is a global variable and can be accessed inside the myfunc function.

Example: Local Variable with the Same Name Inside a Function

x = "awesome"
def myfunc():
  x = "fantastic"
  print("Python is " + x)
myfunc()
print("Python is " + x)

In this example, there are two variables named x: a global one and a local one inside the myfunc function. The local x only exists within the function and doesn't affect the global x.

Using the global Keyword

To create a global variable inside a function or modify a global variable from within a function, you can use the global keyword. Here's an example:

x = "awesome"
def myfunc():
  global x
  x = "fantastic"
myfunc()
print("Python is " + x)

In this example, the global keyword is used inside the myfunc function to indicate that x refers to the global variable. Consequently, the global x is modified by the function.

Remember that modifying global variables within functions can lead to unexpected behavior, so it should be done with caution.



Python Data Types

In programming, data types are essential because they define what kind of data a variable can hold and what operations can be performed on that data. Python has several built-in data types, each serving a specific purpose. Here's a detailed explanation of Python's data types:

Built-in Data Types

Python has the following data types built-in by default, categorized as follows:

Text Type:

  • str (String): Represents text and is enclosed in single or double quotes, e.g., "Hello, World!".

Numeric Types:

  • int (Integer): Represents whole numbers, e.g., 42.
  • float (Float): Represents floating-point numbers (decimals), e.g., 3.14.
  • complex (Complex): Represents complex numbers, e.g., 1j for √(-1).

Sequence Types:

  • list (List): Represents ordered, mutable (changeable) collections, e.g., [1, 2, 3].
  • tuple (Tuple): Represents ordered, immutable (unchangeable) collections, e.g., (1, 2, 3).
  • range (Range): Represents a sequence of numbers, commonly used for looping.

Mapping Type:

  • dict (Dictionary): Represents a collection of key-value pairs, e.g., {"name": "John", "age": 30}.

Set Types:

  • set (Set): Represents an unordered collection of unique elements, e.g., {"apple", "banana", "cherry"}.
  • frozenset (Frozen Set): Similar to a set but immutable.

Boolean Type:

  • bool (Boolean): Represents a binary value, either True or False.

Binary Types:

  • bytes: Represents a sequence of bytes, e.g., b"Hello".
  • bytearray: Mutable version of bytes.
  • memoryview: Represents a memory view of a byte array.

None Type:

  • NoneType: Represents the absence of a value or a null value.

Getting the Data Type

You can use the type() function to determine the data type of a variable or an object. For example:

x = 5
print(type(x))

This code will output the data type of x, which is <class 'int'>.

Setting the Data Type

In Python, the data type is automatically determined when you assign a value to a variable. Here are examples of how data types are set when assigning values:

x = "Hello World"     # str
x = 20                # int
x = 20.5              # float
x = 1j                # complex
x = ["apple", "banana", "cherry"]   # list
x = ("apple", "banana", "cherry")   # tuple
x = range(6)          # range
x = {"name": "John", "age": 36}     # dict
x = {"apple", "banana", "cherry"}   # set
x = frozenset({"apple", "banana", "cherry"})   # frozenset
x = True              # bool
x = b"Hello"          # bytes
x = bytearray(5)      # bytearray
x = memoryview(bytes(5))   # memoryview
x = None              # NoneType

Setting the Specific Data Type

If you want to specify the data type explicitly, you can use constructor functions, which allow you to create variables of a specific data type. Here are examples:

x = str("Hello World")   # str
x = int(20)              # int
x = float(20.5)          # float
x = complex(1j)          # complex
x = list(("apple", "banana", "cherry"))   # list
x = tuple(("apple", "banana", "cherry"))  # tuple
x = range(6)             # range
x = dict(name="John", age=36)            # dict
x = set(("apple", "banana", "cherry"))   # set
x = frozenset(("apple", "banana", "cherry"))  # frozenset
x = bool(5)              # bool
x = bytes(5)             # bytes
x = bytearray(5)         # bytearray
x = memoryview(bytes(5)) # memoryview

In these examples, the constructor functions explicitly set the data type for the variable x.



Python Numbers

In Python, there are three primary numeric types:

1. int (Integer)

An integer is a whole number, positive or negative, without any decimal part. It can be of unlimited length. Examples:

x = 1
y = 35656222554887711
z = -3255522

2. float (Floating-Point Number)

A floating-point number, or simply a float, can have one or more decimals. Examples:

x = 1.10
y = 1.0
z = -35.59

Floats can also be written in scientific notation with an "e" to indicate the power of 10:

x = 35e3  # 35 * 10^3
y = 12E4  # 12 * 10^4
z = -87.7e100  # -87.7 * 10^100

3. complex (Complex Number)

A complex number is written with a "j" as the imaginary part. It consists of a real and an imaginary part. Examples:

x = 3 + 5j
y = 5j
z = -5j

Verifying Data Types

To determine the data type of any object in Python, you can use the type() function. For example:

x = 1
y = 2.8
z = 1j
print(type(x))  # <class 'int'>
print(type(y))  # <class 'float'>
print(type(z))  # <class 'complex'>

Type Conversion

Python allows you to convert between numeric types using specific conversion methods:

  • int(): Converts to an integer.
  • float(): Converts to a floating-point number.
  • complex(): Converts to a complex number.

Example:

x = 1    # int
y = 2.8  # float
z = 1j   # complex
# Convert from int to float
a = float(x)
# Convert from float to int
b = int(y)
# Convert from int to complex
c = complex(x)
print(a)  # 1.0
print(b)  # 2
print(c)  # (1+0j)
print(type(a))  # <class 'float'>
print(type(b))  # <class 'int'>

Note: You cannot directly convert complex numbers into another numeric type.

Generating Random Numbers

To generate random numbers in Python, you can use the random module. While Python doesn't have a random() function, you can use random.randrange() to generate random integers within a specified range:

import random
print(random.randrange(1, 10))  # Generates a random integer between 1 and 9 (inclusive)

This code imports the random module and then uses random.randrange() to print a random integer between 1 and 9.



Python Casting

In Python, casting is the process of converting one data type into another. Python is an object-oriented language, and it uses constructor functions to cast data from one type to another. Here are some common casting methods:

1. int()

The int() constructor function is used to construct an integer from various sources, including integer literals, float literals (by removing all decimals), or string literals (if the string represents a whole number).

x = int(1)    # x will be 1

z = int("3")  # z will be 3

z = float("3")    # z will be 3.0

x = str("s1")  # x will be 's1'

Casting allows you to change the data type of a variable, which can be useful when you need to perform operations or comparisons between different types of data.



Python Strings

Strings are one of the fundamental data types in Python, used to store and manipulate text. In Python, strings can be enclosed in either single quotation marks (') or double quotation marks ("), and both forms are equivalent.

Example:

print("Hello")
print('Hello')

Assigning a String to a Variable

To store a string in a variable, you can simply assign the variable a name followed by an equal sign and the string value.

Example:

a = "Hello"
print(a)

Multiline Strings

Python allows you to create multiline strings using triple quotes (''' or """). This feature is handy when you need to define strings that span multiple lines.

Example:

a = """Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua."""
print(a)

You can also use triple single quotes for the same purpose:

Example:

a = '''Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua.'''
print(a)

The line breaks in the multiline string are preserved in the result.

Strings are Arrays

Strings in Python are essentially arrays of bytes representing Unicode characters. Unlike some other programming languages, Python doesn't have a dedicated character data type; instead, a single character is treated as a string with a length of 1. You can access individual characters or slices of a string using square brackets [].

Example:

a = "Hello, World!"
print(a[1])  # This will print 'e' (character at position 1)

Looping Through a String

Since strings are essentially arrays of characters, you can loop through the characters in a string using a for loop.

Example:

for x in "banana":
    print(x)

This loop will print each character of the word "banana" on a separate line.

String Length

To find the length (the number of characters) of a string, you can use the len() function.

Example:

a = "Hello, World!"
print(len(a))  # This will print 13 (length of the string)

Checking for Substrings

You can check if a specific substring exists within a string using the in keyword. To check for the absence of a substring, you can use the not in keyword.

Example:

txt = "The best things in life are free!"
print("free" in txt)         # This will print True
print("expensive" not in txt)  # This will print True

These operations cover some of the fundamental concepts of working with strings in Python, including creating, accessing, looping through, and checking strings. Strings are an essential data type for handling text and are used extensively in Python programming.

Python - Modify Strings

In Python, strings are immutable, meaning they cannot be changed after they are created. However, Python provides various built-in methods that allow you to create modified versions of strings without altering the original string. These methods return new string objects.

Upper Case and Lower Case

  • The upper() method returns a new string with all characters in uppercase.
  • The lower() method returns a new string with all characters in lowercase.

Examples:

a = "Hello, World!"
print(a.upper())  # Output: "HELLO, WORLD!"
print(a.lower())  # Output: "hello, world!"

Removing Whitespace

Whitespace refers to spaces, tabs, and newline characters that may appear at the beginning or end of a string. Python provides the strip() method to remove these leading and trailing whitespaces.

Example:

a = " Hello, World! "
print(a.strip())  # Output: "Hello, World!"

Replacing Substrings

The replace() method allows you to create a new string by replacing all occurrences of a specified substring with another substring.

Example:

a = "Hello, World!"
print(a.replace("H", "J"))  # Output: "Jello, World!"

Splitting Strings

You can split a string into a list of substrings using the split() method. It takes an optional argument, the separator, and splits the string wherever the separator is found. By default, it splits on whitespace.

Example:

a = "Hello, World!"
print(a.split(","))  # Output: ['Hello', ' World!']

String Concatenation

String concatenation is the process of combining two or more strings to create a new one. In Python, you can use the + operator to concatenate strings together.

Examples:

a = "Hello"
b = "World"
c = a + b  # Concatenation without space
print(c)   # Output: "HelloWorld"
c = a + " " + b  # Concatenation with a space
print(c)        # Output: "Hello World"

String Formatting

String formatting allows you to create a new string by inserting variables or values into a string template. Python provides the format() method for this purpose.

Example:

age = 36
txt = "My name is John, and I am {} years old."
formatted_txt = txt.format(age)
print(formatted_txt)
# Output: "My name is John, and I am 36 years old."

You can use curly braces {} as placeholders in the string template, and then use the format() method to fill in those placeholders with values.

These string manipulation methods are essential when working with text data in Python. They allow you to create modified versions of strings while keeping the original strings intact.



Python - Escape Characters

In Python, escape characters are used to insert special or "illegal" characters into strings. These characters are called "illegal" because they have a special meaning in Python, and using them directly in a string can cause syntax errors or unexpected behavior. To insert these characters, you use a backslash (\) followed by the character you want to insert. Here are some common escape characters used in Python:

  1. \': Inserts a single quote character within a string.
  2. \": Inserts a double quote character within a string.
  3. \\: Inserts a backslash character within a string.
  4. \n: Inserts a new line character.
  5. \r: Inserts a carriage return character.
  6. \t: Inserts a tab character.
  7. \b: Inserts a backspace character.
  8. \f: Inserts a form feed character.
  9. \ooo: Inserts a character with the octal value ooo.
  10. \xhh: Inserts a character with the hexadecimal value hh.

Example 1: Inserting Single Quote and Double Quote


# Using escape characters to insert single and double quotes
single_quote = 'She said, \'Hello!\''
double_quote = "He replied, \"Hi!\""
print(single_quote)  # Output: She said, 'Hello!'
print(double_quote)  # Output: He replied, "Hi!"

Example 2: Inserting New Line and Tab


# Using escape characters to insert new line and tab
new_line = "Line 1\nLine 2"
tabbed = "First\tSecond"
print(new_line)
# Output:
# Line 1
# Line 2
print(tabbed)  # Output: First    Second

Example 3: Inserting Backslash, Carriage Return, and Octal Value


# Using escape characters to insert backslash, carriage return, and octal value
backslash = "This is a backslash: \\"
carriage_return = "Line 1\rLine 2"
octal_value = "\110\145\154\154\157"  # Octal for 'Hello'
print(backslash)        # Output: This is a backslash: \
print(carriage_return)  # Output: Line 2Line 2
print(octal_value)      # Output: Hello

These escape characters allow you to include special characters within strings without causing syntax errors.



Python String Methods

Python provides a wide range of built-in methods that you can use to manipulate strings. These methods allow you to perform various operations on strings, such as changing their case, searching for substrings, splitting and joining them, and much more. It's important to note that all string methods return new values and do not modify the original string. Here's an overview of some commonly used string methods:

  1. capitalize()

    Description: Converts the first character of the string to uppercase, and the rest to lowercase.

    Example:

            text = "hello world"
result = text.capitalize()
print(result)  # Output: "Hello world"

  2. casefold()

    Description: Converts the entire string to lowercase, even for characters that might not have a direct lowercase equivalent.

    Example:

            text = "İstanbul"
result = text.casefold()

    Description: Returns a centered version of the string within a specified width. You can also provide a fill character to pad the empty spaces.

    Example:

print(result)  # Output: "**Python**"

  4. count(substring)

    Description: Returns the number of times a specified substring occurs in the string.

    Example:

  5. encode(encoding)

    Description: Returns an encoded version of the string using the specified encoding.

    Example:

    Description: Checks if the string ends with a specified suffix and returns True or False.

    Example:

result = text.endswith("world!")
print(result)  # Output: True

  7. expandtabs(tabsize)

    Description: Sets the tab size of the string. It replaces tab characters with spaces.

    Example:

  8. find(substring)

    Description: Searches the string for a specified substring and returns the position of the first occurrence (or -1 if not found).

    Example:

            text = "Python is awesome"
position = text.find("is")

  9. format(*args, **kwargs)

    Description: Formats the string by replacing placeholders {} with values from args or kwargs.

    Example:

            name = "Alice"

result = "My name is {}, and I am {} years old.".format(name, age)
print(result)  # Output: "My name is Alice, and I am 30 years old."

  10. format_map(mapping)

    Description: Similar to format(), but takes a mapping (dictionary) as an argument to replace placeholders.

            data = {'name': 'Bob', 'age': 25}

  11. index(substring)

            text = "Hello"

            text = "12345"

Python Booleans:

Boolean values represent one of two states: True or False. They are fundamental in programming and are used for making decisions and controlling the flow of a program.

Boolean Values:

In Python, you can evaluate expressions to determine whether they are True or False. When you compare two values using operators like <, >, ==, etc., Python evaluates the expression and returns a Boolean result:


print(10 > 9)   # Output: True
print(10 == 9)  # Output: False
print(10 < 9)   # Output: False

You can use these Boolean results to make decisions in your code, such as in an if statement:


a = 200
b = 33
if b > a:
  print("b is greater than a")
else:
  print("b is not greater than a")

In this example, the condition b > a is evaluated, and since it's False, the code in the else block is executed.

Evaluate Values and Variables:

You can use the bool() function to explicitly evaluate any value or variable and get either True or False as the result:


print(bool("Hello"))  # Output: True
print(bool(15))       # Output: True
x = "Hello"
y = 15
print(bool(x))  # Output: True
print(bool(y))  # Output: True

Most Values are True:

In Python, almost any value is evaluated as True if it has some content. For example:

  • Any non-empty string is True.
  • Any number that is not 0 is True.
  • Any non-empty list, tuple, set, or dictionary is True.

print(bool("abc"))                  # Output: True
print(bool(123))                    # Output: True
print(bool(["apple", "cherry"]))    # Output: True

Some Values are False:

There are very few values that evaluate to False, including empty values like (), [], {}, "", 0, and None. Additionally, the value False itself is False.


print(bool(False))  # Output: False
print(bool(None))   # Output: False
print(bool(0))      # Output: False
print(bool(""))     # Output: False
print(bool(()))     # Output: False
print(bool([]))     # Output: False
print(bool({}))     # Output: False

In some cases, objects of custom classes can also evaluate to False if the class defines a __len__ function that returns 0 or False.


class myclass():
  def __len__(self):
    return 0
myobj = myclass()
print(bool(myobj))  # Output: False

Functions can Return a Boolean:

You can create functions that return Boolean values. For example:


def myFunction():
  return True
print(myFunction())  # Output: True

These functions can be used to perform specific tasks based on the Boolean result:


def myFunction():
  return True
if myFunction():
  print("YES!")
else:
  print("NO!")

In this case, the function myFunction returns True, so "YES!" is printed.

Built-in Functions Return Booleans:

Python includes many built-in functions that return Boolean values. For example, the isinstance() function can be used to check if an object is of a certain data type and returns True or False accordingly:


x = 200
print(isinstance(x, int))  # Output: True

This code checks if x is an integer and returns True.

Operator Precedence

Operator precedence describes the order in which operations are performed.

For example, parentheses have the highest precedence, meaning that expressions inside parentheses must be evaluated first:


print((6 + 3) - (6 + 3))

Multiplication (*) has higher precedence than addition (+), and therefore multiplications are evaluated before additions:


print(100 + 5 * 3)

The precedence order is described in the table below:

Operator Description
() Parentheses

== != > >= < <= is is not in not in Comparisons, identity, and membership operators

If two operators have the same precedence, the expression is evaluated from left to right.

For example, addition (+) and subtraction (-) have the same precedence:


print(5 + 4 - 7 + 3)

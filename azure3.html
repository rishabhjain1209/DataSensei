Lesson 1: Creating and Managing VMs

In this lesson, you'll learn how to create and manage Azure Virtual Machines (VMs). VMs in Azure offer flexibility and scalability for running various workloads.

Code Example (Azure CLI):

You can create a VM using the Azure CLI. Replace the placeholders with your specific information.


# Create a VM
az vm create --resource-group myResourceGroup --name myVM --image UbuntuLTS --admin-username azureuser --generate-ssh-keys

Lesson 2: Azure VM Storage

Azure VMs rely on Azure Storage for their virtual hard disks (VHDs). Understanding Azure Storage is crucial for optimizing VM performance and durability.

Code Example (Azure PowerShell):

You can create a storage account for your VM using Azure PowerShell:


# Create a storage account
New-AzStorageAccount -ResourceGroupName myResourceGroup -Name mystorageaccount -Location eastus -SkuName Standard_LRS

Lesson 3: VM Networking

Azure VMs need network configurations for communication and security. This lesson covers aspects like virtual networks, subnets, and network security groups.

Code Example (Azure CLI):

Create a virtual network and a subnet using the Azure CLI:


# Create a virtual network
az network vnet create --resource-group myResourceGroup --name myVNet --address-prefix 10.0.0.0/16 --subnet-name mySubnet --subnet-prefix 10.0.0.0/24

Lesson 4: VM Backup and Recovery

Azure offers robust backup and recovery options for VMs. You'll learn about Azure Backup and how to protect your VMs.

Code Example (Azure CLI):

Enable backup for a VM using the Azure CLI:


# Enable backup for a VM
az backup protection enable-for-vm --resource-group myResourceGroup --vault-name myBackupVault --vm myVM

These lessons provide essential knowledge for working with Azure Virtual Machines (IaaS), from creating and managing VMs, understanding Azure VM storage, configuring VM networking, and implementing VM backup and recovery strategies. The provided code examples demonstrate how to perform these tasks using Azure CLI or PowerShell.

Data Types and Operations in R:

Numeric Data Type:

Numeric data types in R are used to represent numbers, both integers and floating-point numbers.


# Numeric Example
x <- 5         # Assigning an integer value to x
y <- 3.14      # Assigning a floating-point value to y
# Basic Arithmetic Operations
sum_result <- x + y   # Addition
diff_result <- x - y  # Subtraction
prod_result <- x * y  # Multiplication
div_result <- x / y   # Division
# Printing Results
print(sum_result)
print(diff_result)
print(prod_result)
print(div_result)

Character Data Type:

Character data types in R are used to store text and are enclosed in single or double quotes.


# Character Example
name <- "Alice"   # Assigning a character value to name
greeting <- 'Hello, World!'  # Assigning a character value to greeting
# Concatenation
full_greeting <- paste(greeting, name)
# Printing Results
print(name)
print(greeting)
print(full_greeting)

Logical Data Type:

Logical data types in R are used to represent binary values - either TRUE or FALSE. They are often used in conditional statements and logical operations.


# Logical Example
is_raining <- TRUE  # Assigning a logical value to is_raining
is_sunny <- FALSE   # Assigning a logical value to is_sunny
# Logical Operations
both_true <- is_raining & is_sunny   # Logical AND
either_true <- is_raining | is_sunny  # Logical OR
not_true <- !is_raining              # Logical NOT
# Printing Results
print(is_raining)
print(is_sunny)
print(both_true)
print(either_true)
print(not_true)

Arithmetic and Logical Operations:

Arithmetic Operations:


# Arithmetic Operations Example
x <- 10
y <- 3
# Addition
addition_result <- x + y  # 10 + 3 = 13
# Subtraction
subtraction_result <- x - y  # 10 - 3 = 7
# Multiplication
multiplication_result <- x * y  # 10 * 3 = 30
# Division
division_result <- x / y  # 10 / 3 = 3.333...
# Exponentiation
exponentiation_result <- x^y  # 10^3 = 1000
# Modulus (Remainder)
modulus_result <- x %% y  # 10 %% 3 = 1
# Printing Results
print(addition_result)
print(subtraction_result)
print(multiplication_result)
print(division_result)
print(exponentiation_result)
print(modulus_result)

Logical Operations:


# Logical Operations Example
a <- TRUE
b <- FALSE
# Logical AND
and_result <- a & b  # TRUE if both a and b are TRUE, otherwise FALSE
# Logical OR
or_result <- a | b   # TRUE if either a or b is TRUE, otherwise FALSE
# Logical NOT
not_a_result <- !a   # Negation of a (i.e., NOT a)
# Logical XOR (Exclusive OR)
xor_result <- xor(a, b)  # TRUE if one of a or b is TRUE, but not both
# Printing Results
print(and_result)
print(or_result)
print(not_a_result)
print(xor_result)

Understanding and using these data types and operations are fundamental when working with data in R. They play a crucial role in data manipulation, conditional statements, and more advanced data analysis tasks.

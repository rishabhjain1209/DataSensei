Module 3: Working with Data

Importing and Exporting Data:

MATLAB provides various functions and tools to import and export data from a variety of file formats, such as text files, spreadsheets, and more. Common functions for data import and export include:

  1. load and save: You can use the load function to load data from MAT-files and the save function to save MATLAB variables to MAT-files.
  2. readtable and writetable: These functions are used to read and write data to and from spreadsheet files, such as CSV, Excel, and more, using MATLAB tables.
  3. importdata: This versatile function can read data from various file formats, such as text, CSV, and Excel files, and returns the data as a structured array.
  4. xlsread and xlswrite: These functions allow you to read and write data to Excel files directly.
  5. fread and fwrite: These functions are used for reading and writing binary data files.
  6. textscan: It is useful for reading data from text files with a specified format.

Data Manipulation and Preprocessing:

MATLAB provides a wide range of functions and tools for data manipulation and preprocessing. Some commonly used functions include:

  • Indexing and Slicing: You can access and modify specific elements or portions of arrays and matrices using indexing and slicing operations.
  • Filtering and Sorting: MATLAB provides functions for filtering data based on conditions and for sorting data in ascending or descending order.
  • Data Transformation: You can perform various data transformations, such as scaling, normalization, and smoothing, using built-in functions.
  • Data Aggregation: Functions like mean, sum, max, and min allow you to aggregate data in arrays and matrices.
  • Reshaping Data: You can reshape arrays and matrices using functions like reshape and permute.

Matrix Operations and Element-Wise Calculations:

MATLAB is well-known for its powerful matrix capabilities. You can perform matrix operations, such as matrix multiplication (*), element-wise multiplication (.*), and more. For example:


A = [1, 2; 3, 4];
B = [5, 6; 7, 8];
C = A * B; % Matrix multiplication
D = A .* B; % Element-wise multiplication

You can also use functions like transpose and inv for matrix transposition and inversion.

Handling Missing Data:

Handling missing data is crucial in data analysis. In MATLAB, missing data is typically represented using the NaN (Not-a-Number) value. You can use various functions and techniques to handle missing data, including:

  1. isnan: This function checks for NaN values in arrays and matrices.
  2. fillmissing: The fillmissing function allows you to fill missing data using various methods, such as filling with a constant, linear interpolation, or mean/mode values.
  3. Data Interpolation: MATLAB provides interpolation functions like interp1 for 1-D data and interp2 for 2-D data to estimate missing values based on surrounding data points.
  4. Data Removal: You can remove rows or columns containing missing data using logical indexing and the isnan function.
  5. Statistical Imputation: You can impute missing values using statistical techniques like mean imputation, median imputation, or regression imputation.

Overall, MATLAB's data manipulation and preprocessing capabilities, combined with its matrix operations, make it a powerful tool for handling and preparing data for analysis, modeling, and visualization.

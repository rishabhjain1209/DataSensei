Python Functions

Creating a Function

In Python, a function is defined using the def keyword.

def greet():
    print("Hello from a function")

Calling a Function

To execute a function, you use its name followed by parentheses.

greet()

Function Parameters

Functions can accept input data known as parameters or arguments.

def greet_with_name(name):
    print("Hello, " + name)

Function Arguments

Arguments are the actual values you pass to a function when calling it.

greet_with_name("Alice")

Number of Arguments

A function must be called with the correct number of arguments that match its parameter list.

def add(a, b):
    result = a + b
    print("Sum:", result)
add(5, 3)

Returning Values

Functions can return values using the return statement.

def multiply(x, y):
    result = x * y
    return result
product = multiply(4, 6)
print("Product:", product)

Default Values for Parameters

You can provide default values for function parameters.

def greet_with_default(name="User"):
    print("Hello, " + name)
greet_with_default()
greet_with_default("Alice")

Variable-Length Argument Lists

Python allows functions to accept a variable number of arguments using *args.

def sum_values(*args):
    total = sum(args)
    return total
sum1 = sum_values(1, 2, 3)
sum2 = sum_values(10, 20, 30, 40)

Keyword Arguments

You can pass arguments by keyword, specifying which parameter each argument corresponds to.

def display_info(name, age, city):
    print(f"Name: {name}, Age: {age}, City: {city}")
display_info(age=25, name="Alice", city="New York")


Python Function Concepts

Arbitrary Arguments - *args

When you don't know how many arguments will be passed into your function, you can use *args as a parameter.

def my_function(*kids):
    print("The youngest child is " + kids[2])
my_function("Emil", "Tobias", "Linus")

Keyword Arguments

Python allows you to send arguments with the key=value syntax, which means you can pass arguments in any order.

def my_function(child3, child2, child1):
    print("The youngest child is " + child3)
my_function(child1="Emil", child2="Tobias", child3="Linus")

Arbitrary Keyword Arguments - **kwargs

When you don't know how many keyword arguments will be passed, you can use **kwargs as a parameter.

def my_function(**kid):
    print("His last name is " + kid["lname"])
my_function(fname="Tobias", lname="Refsnes")

Default Parameter Value

You can set default values for parameters in a function.

def my_function(country="Norway"):
    print("I am from " + country)
my_function("Sweden")
my_function("India")
my_function()
my_function("Brazil")

Passing a List as an Argument

You can send various data types as arguments to a function, including lists.

def my_function(food):
    for x in food:
        print(x)
fruits = ["apple", "banana", "cherry"]
my_function(fruits)

Return Values

To make a function return a value, you use the return statement.

def my_function(x):
    return 5 * x
result1 = my_function(3)
result2 = my_function(5)
result3 = my_function(9)
print(result1)  # Output: 15
print(result2)  # Output: 25
print(result3)  # Output: 45

Recursion

Python supports function recursion, which means a function can call itself.

def tri_recursion(k):
    if k > 0:
        result = k + tri_recursion(k - 1)
        print(result)
    else:
        result = 0
    return result
print("\nRecursion Example Results:")
tri_recursion(6)

The pass Statement

A function definition cannot be empty, but you can use the pass statement as a placeholder.

def my_function():
    pass

The pass statement indicates that the function does nothing and can be implemented later.



Python Lambda Functions

What is a Lambda Function?

A lambda function is a small, anonymous function in Python. It can take any number of arguments but can only have one expression. Lambda functions are often used when you need a simple function for a short period and don't want to define a full function using def.

Syntax of a Lambda Function

The syntax of a lambda function is as follows:

lambda arguments: expression

The expression is executed, and the result is returned.

Code Examples

Example 1: Lambda with One Argument

In this example, we define a lambda function x that takes one argument a and adds 10 to it. Then, we call the lambda function with an argument of 5 and print the result.

x = lambda a: a + 10
print(x(5))  # Output: 15

Example 2: Lambda with Two Arguments

Here, we define a lambda function x that takes two arguments a and b and multiplies them. We call the lambda function with arguments 5 and 6 and print the result.

x = lambda a, b: a * b
print(x(5, 6))  # Output: 30

Example 3: Lambda with Three Arguments

In this example, we define a lambda function x that takes three arguments a, b, and c and calculates their sum. We call the lambda function with arguments 5, 6, and 2 and print the result.

x = lambda a, b, c: a + b + c
print(x(5, 6, 2))  # Output: 13

Why Use Lambda Functions?

Lambda functions are especially useful when you need an anonymous function for a short period. They are often used inside other functions. Here's an example:

def myfunc(n):
    return lambda a: a * n
mydoubler = myfunc(2)
print(mydoubler(11))  # Output: 22
mytripler = myfunc(3)
print(mytripler(11))  # Output: 33

In this code, myfunc is a function that returns a lambda function. Depending on the argument passed to myfunc, you get a function that can double or triple a number. Lambda functions inside myfunc simplify the creation of these functions on the fly.

Lambda functions are convenient for short, simple operations, but for more complex logic, it's better to use named functions defined with def.



Using Python Lists as Arrays

Arrays in Python

Python doesn't have built-in support for arrays like some other programming languages do. Instead, Python uses lists, which can serve as arrays. To work with arrays in a more specialized way, you may need to import libraries like NumPy.

Creating an Array (List)

You can create an array-like structure using Python lists. Lists can hold multiple values in a single variable.

cars = ["Ford", "Volvo", "BMW"]

Accessing Elements of an Array (List)

You access elements in a list by using their index, which starts from 0.

x = cars[0]  # x will be "Ford"
    
cars[0] = "Toyota"  # Modifying the value of the first array item

Finding the Length of an Array (List)

You can use the len() method to find the number of elements in a list.

x = len(cars)  # x will be 3

Looping Through Array (List) Elements

You can use a for loop to iterate through all the elements of a list.

for x in cars:
    print(x)

Adding Elements to an Array (List)

To add elements to the end of a list, you can use the append() method.

cars.append("Honda")

Removing Elements from an Array (List)

You can remove elements from a list using methods like pop() and remove().

cars.pop(1)  # Deletes the second element in the Cars list
    
cars.remove("Volvo")  # Deletes the element with the value "Volvo"

Note: Lists in Python are versatile and can be used for various purposes, including simulating arrays. However, if you need more advanced array operations or numerical computations, you may want to explore libraries like NumPy.



Array Methods in Python

Python provides a set of built-in methods that you can use to manipulate lists (which can serve as arrays).

1. append() - Adds an Element at the End of the List

This method appends an element to the end of the list.

fruits = ["apple", "banana", "cherry"]
fruits.append("date")
# fruits will now be ["apple", "banana", "cherry", "date"]

2. clear() - Removes All Elements from the List

The clear() method removes all elements from the list, leaving it empty.

fruits = ["apple", "banana", "cherry"]
fruits.clear()
# fruits will now be an empty list: []

3. copy() - Returns a Copy of the List

The copy() method returns a new list that is a copy of the original list.

fruits = ["apple", "banana", "cherry"]
fruits_copy = fruits.copy()
# fruits_copy is a copy of fruits

4. count() - Returns the Number of Elements with the Specified Value

This method counts the number of times a specified element appears in the list.

fruits = ["apple", "banana", "cherry", "apple"]
count = fruits.count("apple")
# count will be 2 (there are two "apple" elements in the list)

5. extend() - Adds Elements of an Iterable to the End of the List

The extend() method adds the elements of an iterable (e.g., another list) to the end of the current list.

fruits = ["apple", "banana", "cherry"]
more_fruits = ["date", "fig"]
fruits.extend(more_fruits)
# fruits will now be ["apple", "banana", "cherry", "date", "fig"]

6. index() - Returns the Index of the First Element with the Specified Value

This method returns the index of the first occurrence of a specified element in the list.

fruits = ["apple", "banana", "cherry"]
index = fruits.index("banana")
# index will be 1 (the index of "banana" in the list)

7. insert() - Adds an Element at the Specified Position

The insert() method adds an element at the specified position in the list.

fruits = ["apple", "banana", "cherry"]
fruits.insert(1, "date")
# fruits will now be ["apple", "date", "banana", "cherry"]

8. pop() - Removes the Element at the Specified Position

The pop() method removes and returns the element at the specified position in the list.

fruits = ["apple", "banana", "cherry"]
removed_fruit = fruits.pop(1)
# removed_fruit will be "banana", and fruits will be ["apple", "cherry"]

9. remove() - Removes the First Item with the Specified Value

The remove() method removes the first occurrence of a specified element from the list.

fruits = ["apple", "banana", "cherry", "apple"]
fruits.remove("apple")
# fruits will now be ["banana", "cherry", "apple"]

10. reverse() - Reverses the Order of the List

The reverse() method reverses the order of elements in the list.

fruits = ["apple", "banana", "cherry"]
fruits.reverse()
# fruits will now be ["cherry", "banana", "apple"]

11. sort() - Sorts the List

The sort() method sorts the elements of the list in ascending order (for numeric values) or lexicographic order (for strings).

numbers = [3, 1, 2, 5, 4]
numbers.sort()
# numbers will now be [1, 2, 3, 4, 5]

These array methods provide powerful ways to manipulate lists (arrays) in Python, making it easier to work with collections of data.
